# Python进阶 - 课程结构说明

## 模块概述

本模块是AI学习体系的基础模块，旨在帮助学习者从Python基础水平提升到能够熟练使用Python进行AI开发的水平。

## 学习目标

1. 掌握面向对象编程的核心概念和实践
2. 理解Python的高级特性（装饰器、生成器、上下文管理器等）
3. 熟练使用常用库进行数据处理和网络请求
4. 能够独立完成中等复杂度的Python项目

## 模块结构

### 01_面向对象编程
- **核心内容**：类、对象、继承、多态、特殊方法
- **学习时间**：2-3周
- **难度**：中等

### 02_高级特性
- **核心内容**：装饰器、生成器、上下文管理器、元编程、并发编程
- **学习时间**：2-3周
- **难度**：较难

### 03_常用库
- **核心内容**：requests、json/csv处理、正则表达式、其他工具库
- **学习时间**：1-2周
- **难度**：简单到中等

### 04_项目实战
- **项目数量**：8个（2简单 + 2中等 + 2进阶 + 2综合）
- **学习时间**：3-4周
- **难度**：递增

## 学习建议

1. **循序渐进**：按照模块顺序学习，不要跳跃
2. **多实践**：每学一个概念，立即写代码
3. **做项目**：通过项目巩固知识
4. **查文档**：学会查阅Python官方文档

## 前置要求

- Python基础语法（变量、数据类型、控制流）
- 函数定义和使用
- 基本的数据结构（列表、字典）

## 后续课程

完成本模块后，可以进入：
- 02_数学基础
- 03_数据处理基础
