## 📋 项目简介

这是一个**全面、系统、前沿**的人工智能学习体系，旨在帮助学习者从Python基础成长为人工智能领域的全能大师。

### 🎯 项目目标

- **学习目标**：提供从零基础到AI专家的完整学习路径
- **教育目标**：适合初学者/小白，内容详细易懂，循序渐进
- **实践目标**：理论与实践并重，所有代码可运行，所有项目可复现
- **前沿目标**：覆盖2023-2025年最新AI技术和研究成果

### 👥 目标学习者

- **Python基础水平**：简单语法、基本数据类型、控制流
- **学习背景**：初学者/小白，目标是成为AI领域大师
- **学习时间**：1-3年（不限）
- **学习要求**：全面、前沿、高质量、详细易懂

## 🚀 快速开始

### 环境要求

- **Python**: 3.8+
- **操作系统**: Windows / macOS / Linux
- **推荐工具**: Jupyter Notebook / JupyterLab

### 安装步骤

1. **克隆仓库**

```bash
git clone https://github.com/Upstee/AI_learning_path.git
cd 人工智能学习
```

2. **安装依赖**

```bash
# 安装基础依赖
pip install numpy pandas matplotlib jupyter

# 安装深度学习框架（根据需要选择）
pip install torch torchvision  # PyTorch
# 或
pip install tensorflow  # TensorFlow
```

3. **启动Jupyter Notebook**

```bash
jupyter notebook
# 或
jupyter lab
```

4. **开始学习**

- 从 `00_开始_AI全景导览/` 开始
- 按照学习路径顺序学习
- 每个模块都有详细的README和代码示例

### 学习建议

1. **循序渐进**：严格按照基础阶段顺序学习
2. **动手实践**：所有代码都要自己运行一遍
3. **完成练习**：完成每个模块的练习题
4. **做项目**：完成实战项目，巩固知识
5. **自我评估**：完成自我评估，确保掌握

## 📖 使用指南

### 每个模块的标准结构

每个学习模块都包含以下8个部分：

1. **课程概述** - 课程目标、学习时间、难度等级
2. **前置知识检查** - 必备前置概念、入门小测
3. **核心知识点详解** - 理论推导、数学公式、图解
