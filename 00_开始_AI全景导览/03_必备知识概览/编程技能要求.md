## 二、面向对象编程（⭐⭐⭐⭐ 重要）

### 2.1 类和对象

#### 必须掌握
- **类定义**：
  - class关键字
  - __init__方法（构造函数）
  - self参数
  - 实例变量
  - 实例方法
  
- **对象创建**：
  - 实例化
  - 访问属性和方法

#### 代码示例
```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        self.scores = []
    
    def add_score(self, score):
        self.scores.append(score)
    
    def get_average(self):
        if not self.scores:
            return 0
        return sum(self.scores) / len(self.scores)

# 使用
student = Student("张三", 20)
student.add_score(85)
student.add_score(90)
print(student.get_average())
```

### 2.2 继承

#### 必须掌握
- **单继承**：
  - 子类继承父类
  - super()函数
  - 方法重写
  
- **多继承**：
  - 多个父类
  - 方法解析顺序（MRO）

#### 代码示例
```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        return f"我是{self.name}，{self.age}岁"

class Student(Person):
    def __init__(self, name, age, student_id):
        super().__init__(name, age)
        self.student_id = student_id
    
    def introduce(self):
        return f"{super().introduce()}，学号是{self.student_id}"
```

### 2.3 特殊方法

#### 应该掌握
- **常用特殊方法**：
  - __str__：字符串表示
  - __repr__：对象表示
  - __len__：长度
  - __getitem__：索引访问
  - __call__：可调用对象

#### 代码示例
```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __str__(self):
        return f"Vector({self.x}, {self.y})"
    
    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)
    
    def __len__(self):
        return 2
```

## 三、高级特性（⭐⭐⭐ 应该了解）

### 3.1 装饰器

#### 应该了解
- **装饰器定义**：
  - @语法糖
  - 函数装饰器
  - 类装饰器
  
- **常用装饰器**：
  - @property
  - @staticmethod
  - @classmethod

#### 代码示例
```python
def timer(func):
    def wrapper(*args, **kwargs):
        import time
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__} took {end - start:.2f} seconds")
        return result
    return wrapper

@timer
def slow_function():
    import time
    time.sleep(1)
    return "Done"
```

### 3.2 生成器

#### 应该了解
- **生成器函数**：
  - yield关键字
  - 惰性计算
  - 内存效率
  
- **生成器表达式**：
  - 类似列表推导式
  - 使用()而不是[]

#### 代码示例
```python
# 生成器函数
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# 使用
for num in fibonacci(10):
    print(num)

# 生成器表达式
squares = (x**2 for x in range(10))
```

  - 自动资源管理

  - @contextmanager装饰器
