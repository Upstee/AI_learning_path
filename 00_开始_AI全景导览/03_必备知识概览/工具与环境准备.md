## 一、开发环境

### 1.1 Python安装

#### Windows系统

**步骤1：下载Python**
- 访问 https://www.python.org/downloads/
- 下载Python 3.8或更高版本（推荐3.10+）
- 选择Windows installer (64-bit)

**步骤2：安装Python**
- 运行安装程序
- **重要**：勾选"Add Python to PATH"
- 选择"Install Now"或"Customize installation"
- 等待安装完成

**步骤3：验证安装**
```bash
# 打开命令提示符（CMD）或PowerShell
python --version
# 应该显示：Python 3.x.x

pip --version
# 应该显示：pip版本信息
```

#### macOS系统

**方法1：使用Homebrew（推荐）**
```bash
# 安装Homebrew（如果还没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装Python
brew install python3

# 验证
python3 --version
pip3 --version
```

**方法2：从官网下载**
- 访问 https://www.python.org/downloads/
- 下载macOS安装包
- 运行安装程序

#### Linux系统

**Ubuntu/Debian**：
```bash
sudo apt update
sudo apt install python3 python3-pip
```

**CentOS/RHEL**：
```bash
sudo yum install python3 python3-pip
```

### 1.2 包管理工具

#### pip（Python包管理器）

**基本使用**：
```bash
# 安装包
pip install package_name

# 安装特定版本
pip install package_name==1.0.0

# 升级包
pip install --upgrade package_name

# 卸载包
pip uninstall package_name

# 列出已安装的包
pip list

# 导出依赖
pip freeze > requirements.txt

# 从requirements.txt安装
pip install -r requirements.txt
```

**常用AI包**：
```bash
# 数据处理
pip install numpy pandas matplotlib seaborn

# 机器学习
pip install scikit-learn

# 深度学习
pip install torch torchvision
# 或
pip install tensorflow

# Jupyter
pip install jupyter notebook

# 其他工具
pip install requests beautifulsoup4
```

#### conda（Anaconda/Miniconda）

**安装Anaconda**：
- 访问 https://www.anaconda.com/products/distribution
- 下载适合你系统的安装包
- 运行安装程序

**基本使用**：
```bash
# 创建环境
conda create -n ai_env python=3.10

# 激活环境
conda activate ai_env

# 安装包
conda install numpy pandas

# 从conda-forge安装
conda install -c conda-forge package_name

# 列出环境
conda env list

# 导出环境
conda env export > environment.yml

# 从environment.yml创建环境
conda env create -f environment.yml
```

### 1.3 虚拟环境

#### venv（Python内置）

**创建虚拟环境**：
```bash
# 创建虚拟环境
python -m venv ai_env

# Windows激活
ai_env\Scripts\activate

# macOS/Linux激活
source ai_env/bin/activate

# 停用
deactivate
```

#### virtualenv

**安装和使用**：
```bash
# 安装
pip install virtualenv

# 创建环境
virtualenv ai_env

# 激活（同venv）
source ai_env/bin/activate  # macOS/Linux
ai_env\Scripts\activate     # Windows
```

## 六、快速开始脚本

### 6.1 环境搭建脚本

**setup_env.sh**（macOS/Linux）：
```bash
#!/bin/bash

# 创建虚拟环境
python3 -m venv ai_env
source ai_env/bin/activate

# 升级pip
pip install --upgrade pip

# 安装基础包
pip install numpy pandas matplotlib seaborn
pip install scikit-learn
pip install jupyter notebook
pip install requests

# 创建项目目录
mkdir -p projects notebooks data

echo "环境搭建完成！"
echo "激活环境：source ai_env/bin/activate"
```

**setup_env.bat**（Windows）：
```batch
@echo off

REM 创建虚拟环境
python -m venv ai_env
call ai_env\Scripts\activate.bat

REM 升级pip
python -m pip install --upgrade pip

REM 安装基础包
pip install numpy pandas matplotlib seaborn
pip install scikit-learn
pip install jupyter notebook
pip install requests

REM 创建项目目录
mkdir projects
mkdir notebooks
mkdir data

echo 环境搭建完成！
echo 激活环境：ai_env\Scripts\activate
```

### 6.2 验证脚本

**check_env.py**：
```python
#!/usr/bin/env python3
"""检查环境配置"""

import sys

def check_python():
    """检查Python版本"""
    version = sys.version_info
    if version.major >= 3 and version.minor >= 8:
        print(f"✓ Python {version.major}.{version.minor}.{version.micro}")
        return True
    else:
        print(f"✗ Python版本过低：{version.major}.{version.minor}")
        return False

def check_package(package_name, import_name=None):
    """检查包是否安装"""
    if import_name is None:
        import_name = package_name
    
    try:
        __import__(import_name)
        print(f"✓ {package_name} 已安装")
        return True
    except ImportError:
        print(f"✗ {package_name} 未安装")
        return False

def main():
    """主函数"""
    print("=== 环境检查 ===\n")
    
    # 检查Python
    python_ok = check_python()
    
    # 检查包
    packages = [
        ("numpy", "numpy"),
        ("pandas", "pandas"),
        ("matplotlib", "matplotlib"),
        ("scikit-learn", "sklearn"),
        ("jupyter", "jupyter"),
    ]
    
    print("\n=== 包检查 ===")
    all_ok = python_ok
    for package, import_name in packages:
        if not check_package(package, import_name):
            all_ok = False
    
    print("\n=== 检查结果 ===")
    if all_ok:
        print("✓ 环境配置完整！")
    else:
        print("✗ 部分环境未配置，请安装缺失的包")

if __name__ == "__main__":
    main()
```

## 七、常见问题

### Q1：Python安装后无法使用？
**A**：
- Windows：检查是否勾选"Add Python to PATH"
- 使用`python3`而不是`python`（macOS/Linux）
- 检查环境变量PATH

### Q2：pip安装包很慢？
**A**：
- 使用国内镜像源：
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple package_name
```
- 或配置永久镜像源

### Q3：虚拟环境激活失败？
**A**：
- Windows：使用`Scripts\activate`而不是`bin/activate`
- 检查虚拟环境路径
- 确保使用正确的Python版本创建环境

### Q4：Jupyter无法启动？
**A**：
- 检查是否安装：`pip list | grep jupyter`
- 重新安装：`pip install --upgrade jupyter`
- 检查端口是否被占用

### Q5：Git配置问题？
**A**：
- 检查Git是否安装：`git --version`
- 配置用户名和邮箱
- 检查网络连接（GitHub访问）
