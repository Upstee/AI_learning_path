1. **Clone Repository**


2. **Install Dependencies**

```bash
# Install basic dependencies
pip install numpy pandas matplotlib jupyter

# Install deep learning framework (choose as needed)

```

3. **Start Jupyter Notebook**

```bash
# or

```

- Start from `00_AI_Overview/`
