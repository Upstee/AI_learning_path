# 简单项目1：基因聚类分析

## 项目描述

使用层次聚类对基因表达数据进行聚类分析，识别相似的基因。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备基因表达数据（模拟或真实）
   - 数据预处理和标准化

2. **模型训练**
   - 使用层次聚类
   - 选择合适的链接方法
   - 绘制树状图

3. **结果分析**
   - 分析基因簇
   - 可视化聚类结果
   - 生成分析报告

- `main.py` - 主程序
