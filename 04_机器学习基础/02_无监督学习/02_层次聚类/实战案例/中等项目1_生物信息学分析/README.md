# 中等项目1：生物信息学分析

## 项目描述

使用层次聚类进行生物信息学分析，如蛋白质分类、物种分类等。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据预处理**
   - 生物数据加载和清洗
   - 特征工程
   - 数据标准化

2. **模型训练**
   - 训练层次聚类器
   - 选择合适的链接方法
   - 参数调优

3. **结果分析**
   - 分析生物分类结果
   - 可视化树状图
   - 生成分析报告

## 项目文件

- `main.py` - 主程序
- `analysis.py` - 分析模块
- `README.md` - 项目说明（本文件）
