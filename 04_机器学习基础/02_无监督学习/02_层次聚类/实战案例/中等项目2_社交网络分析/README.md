# 中等项目2：社交网络分析

## 项目描述

使用层次聚类分析社交网络，识别社区结构。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **网络数据预处理**
   - 构建网络图
   - 提取节点特征
   - 计算相似度

2. **模型训练**
   - 使用层次聚类
   - 选择合适的距离度量
   - 绘制树状图

3. **结果分析**
   - 分析社区结构
   - 可视化网络和聚类结果

## 项目文件

- `main.py` - 主程序
- `network.py` - 网络处理模块
- `README.md` - 项目说明（本文件）
