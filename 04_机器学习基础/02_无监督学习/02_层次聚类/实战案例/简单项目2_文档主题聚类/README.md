# 简单项目2：文档主题聚类

## 项目描述

使用层次聚类对文档进行主题聚类，将文档分成不同的主题。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **文本预处理**
   - 文本清洗和分词
   - 特征提取（TF-IDF）

2. **模型训练**
   - 使用层次聚类
   - 选择合适的链接方法
   - 绘制树状图

3. **结果分析**
   - 分析每个主题的文档
   - 可视化聚类结果
