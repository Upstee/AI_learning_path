# 中等项目1：地理热点分析

## 项目描述

使用DBSCAN分析地理数据，识别热点区域（如犯罪热点、商业热点等）。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据预处理**
   - 地理数据加载和清洗
   - 坐标转换（如果需要）
   - 数据标准化

2. **模型训练**
   - 使用DBSCAN聚类
   - 选择合适的参数
   - 识别热点区域

3. **结果分析**
   - 分析热点区域特征
   - 可视化地理分布
   - 生成热点报告

## 项目文件

- `main.py` - 主程序
- `analysis.py` - 分析模块
- `README.md` - 项目说明（本文件）
