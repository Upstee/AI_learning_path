# 简单项目1：异常检测系统

## 项目描述

使用DBSCAN构建一个简单的异常检测系统，识别数据中的异常点。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备包含异常值的数据
   - 数据预处理和标准化

2. **模型训练**
   - 使用DBSCAN进行异常检测
   - 选择合适的参数
   - 识别异常点

3. **结果展示**
   - 可视化异常点
   - 分析异常点特征
   - 生成异常报告

- `main.py` - 主程序
