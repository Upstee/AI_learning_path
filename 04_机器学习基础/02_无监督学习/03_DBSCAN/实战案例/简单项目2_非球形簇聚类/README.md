# 简单项目2：非球形簇聚类

## 项目描述

使用DBSCAN对非球形簇进行聚类，展示DBSCAN处理复杂形状的能力。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 生成或使用非球形数据（如月牙形、环形）
   - 数据预处理

2. **模型训练**
   - 使用DBSCAN聚类
   - 选择合适的参数
   - 对比K-means的结果

3. **结果展示**
   - 可视化聚类结果
   - 分析DBSCAN的优势
