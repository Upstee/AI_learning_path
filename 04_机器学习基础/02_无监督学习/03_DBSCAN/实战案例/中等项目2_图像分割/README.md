# 中等项目2：图像分割

## 项目描述

使用DBSCAN对图像进行分割，将图像分成不同的区域。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **图像处理**
   - 加载图像
   - 提取像素特征（颜色、位置）
   - 数据预处理

2. **模型训练**
   - 使用DBSCAN聚类
   - 选择合适的参数
   - 处理大规模数据（采样）

3. **结果展示**
   - 可视化分割结果
   - 对比不同参数的效果
