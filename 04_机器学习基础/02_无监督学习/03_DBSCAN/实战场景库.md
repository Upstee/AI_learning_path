## 常见应用场景

### 场景1：异常检测

**业务背景**：
- 信用卡欺诈检测
- 网络入侵检测
- 设备故障检测

**实现**：
```python
from sklearn.cluster import DBSCAN

# DBSCAN自动识别异常点
dbscan = DBSCAN(eps=0.5, min_samples=5)
labels = dbscan.fit_predict(X)

# 噪声点就是异常点
anomalies = X[labels == -1]
```

---

### 场景2：地理热点分析

**业务背景**：
- 发现犯罪热点区域
- 发现商业热点
- 发现交通拥堵点

**实现**：
```python
# 使用经纬度坐标
locations = np.array([[lat, lon] for lat, lon in zip(lats, lons)])

dbscan = DBSCAN(eps=0.01, min_samples=5, metric='haversine')
labels = dbscan.fit_predict(locations)

# 可视化热点
plt.scatter(locations[:, 1], locations[:, 0], c=labels, cmap='viridis')
plt.show()
```

---

### 场景3：非球形簇聚类

**业务背景**：
- 处理月牙形、环形等复杂形状的簇
- K-means无法处理的情况

**实现**：
```python
# 生成非球形数据
X, _ = make_moons(n_samples=200, noise=0.05)

# DBSCAN可以处理
dbscan = DBSCAN(eps=0.3, min_samples=5)
labels = dbscan.fit_predict(X)
```
