## 📍 学习前自测：我准备好了吗？

- [ ] **K-means基础**
  - [ ] 理解聚类的基本概念

  - [ ] 如果不熟悉，请先学习：[01_K-means](../01_K-means/)

- [ ] **距离计算**
  - [ ] 理解欧氏距离

- [ ] **Python基础**
  - [ ] 能够使用NumPy计算距离
  - [ ] 能够使用matplotlib绘图
  - [ ] 如果不熟悉，请先学习：[01_Python进阶](../../01_Python进阶/)

### 学习环境检查

- [ ] Python 3.8+ 已安装
- [ ] 已安装必要库：numpy, pandas, matplotlib, scikit-learn
- [ ] 能够运行Jupyter Notebook或Python脚本

### 学习目标明确

- [ ] 我知道为什么要学DBSCAN（处理非球形簇、异常检测）

## 🆘 如果卡住了：怎么办？

### 问题1：不理解密度概念

**解决方法**：
1. 回到快速上手，观察示例
2. 用生活中的例子理解（如人群密度）
3. 查看FAQ

### 问题2：参数选择困难

**解决方法**：
1. 使用k-距离图选择eps
2. min_samples通常设为2-5
3. 多实验，观察效果
