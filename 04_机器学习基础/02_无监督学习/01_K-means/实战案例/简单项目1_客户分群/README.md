# 简单项目1：客户分群

## 项目描述

使用K-means对客户进行分群，分析不同客户群体的特征。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备客户数据（年龄、收入、消费等）
   - 数据预处理和标准化

2. **模型训练**
   - 使用K-means聚类
   - 选择最优K值
   - 训练模型

3. **结果分析**
   - 分析每个客户群体的特征
   - 可视化聚类结果
   - 生成客户画像

- `main.py` - 主程序
