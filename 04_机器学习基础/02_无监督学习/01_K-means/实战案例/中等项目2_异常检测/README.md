# 中等项目2：异常检测

## 项目描述

使用K-means进行异常检测，识别数据中的异常点。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据预处理**
   - 数据加载和清洗
   - 特征工程
   - 数据标准化

2. **异常检测**
   - 使用K-means聚类
   - 计算样本到簇中心的距离
   - 识别异常点

3. **结果分析**
   - 可视化异常点
   - 分析异常原因
   - 生成异常报告
