# 中等项目1：文档聚类

## 项目描述

使用K-means对文档进行聚类，将文档分成不同的主题。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **文本预处理**
   - 文本清洗和分词
   - 特征提取（TF-IDF）
   - 数据预处理

2. **模型训练**
   - 训练K-means聚类器
   - 选择最优K值
   - 参数调优

3. **结果分析**
   - 分析每个簇的主题
   - 可视化聚类结果
   - 生成主题报告

## 项目文件

- `main.py` - 主程序
- `preprocessing.py` - 文本预处理模块
- `README.md` - 项目说明（本文件）
