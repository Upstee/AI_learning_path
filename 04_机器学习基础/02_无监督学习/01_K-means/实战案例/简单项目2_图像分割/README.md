# 简单项目2：图像分割

## 项目描述

使用K-means对图像进行分割，将图像分成K个颜色区域。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **图像处理**
   - 加载图像
   - 将图像转换为特征向量

2. **模型训练**
   - 使用K-means聚类
   - 选择合适的K值

3. **结果展示**
   - 可视化分割结果
   - 对比不同K值的效果
