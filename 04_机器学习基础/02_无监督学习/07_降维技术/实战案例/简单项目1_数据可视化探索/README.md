# 简单项目1：数据可视化探索

## 项目描述

使用降维技术将高维数据降维到2-3维，进行可视化探索。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备高维数据（模拟或真实）
   - 数据预处理和标准化

2. **降维**
   - 使用PCA和t-SNE降维
   - 比较不同降维方法的效果

3. **可视化**
   - 可视化降维后的数据
   - 分析数据结构和模式
   - 生成可视化报告

- `main.py` - 主程序
