# 简单项目2：特征提取

## 项目描述

使用PCA从原始特征中提取主成分作为新特征。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 加载数据
   - 数据预处理

2. **特征提取**
   - 使用PCA提取主成分
   - 选择合适的降维维度

3. **结果分析**
   - 分析主成分特征
   - 对比原始特征和主成分特征

- `main.py` - 主程序
