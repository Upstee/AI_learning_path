# 简单项目1：数据可视化

## 项目描述

使用PCA将高维数据降到2维或3维进行可视化。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 加载高维数据
   - 数据预处理和标准化

2. **PCA降维**
   - 使用PCA降维到2维或3维
   - 选择合适的降维维度

3. **结果展示**
   - 可视化降维后的数据
   - 分析数据分布
   - 生成可视化报告

- `main.py` - 主程序
