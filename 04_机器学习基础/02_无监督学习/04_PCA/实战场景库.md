## 常见应用场景

### 场景1：高维数据可视化

**业务背景**：
- 将高维数据降到2-3维进行可视化
- 探索数据分布和模式

**实现**：
```python
# 高维数据（例如：100维）
pca = PCA(n_components=2)
X_2d = pca.fit_transform(X_high_dim)

# 可视化
plt.scatter(X_2d[:, 0], X_2d[:, 1], c=labels, cmap='viridis')
plt.show()
```

---

### 场景2：特征提取和降维

**业务背景**：
- 减少特征数量，去除噪声
- 加速机器学习训练

**实现**：
```python
# 原始特征：1000维
# PCA降维：50维（保留95%信息）
pca = PCA(n_components=0.95)
X_reduced = pca.fit_transform(X)

# 在降维数据上训练
classifier.fit(X_reduced, y)
```

---

### 场景3：图像压缩

**业务背景**：
- 压缩图像数据
- 减少存储空间

**实现**：
```python
# 图像：28x28 = 784维
# PCA降维：50维
pca = PCA(n_components=50)
X_compressed = pca.fit_transform(images)

# 重建图像
X_reconstructed = pca.inverse_transform(X_compressed)
```
