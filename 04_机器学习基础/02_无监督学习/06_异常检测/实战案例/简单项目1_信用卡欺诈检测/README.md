# 简单项目1：信用卡欺诈检测

## 项目描述

使用异常检测方法检测信用卡交易中的欺诈行为。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备信用卡交易数据（模拟或真实）
   - 数据预处理和标准化

2. **异常检测**
   - 使用Isolation Forest或One-Class SVM
   - 识别异常交易

3. **结果展示**
   - 可视化异常交易
   - 分析异常特征
   - 生成检测报告

- `main.py` - 主程序
