# 简单项目1：数据可视化探索

## 项目描述

使用t-SNE对高维数据进行可视化，探索数据的结构和模式。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 加载高维数据
   - 数据预处理和标准化

2. **t-SNE降维**
   - 使用t-SNE降维到2维
   - 选择合适的参数

3. **结果展示**
   - 可视化降维后的数据
   - 分析数据分布和模式
   - 生成可视化报告

- `main.py` - 主程序
