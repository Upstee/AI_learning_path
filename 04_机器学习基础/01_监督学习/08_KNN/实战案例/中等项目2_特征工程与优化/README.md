# 中等项目2：特征工程与优化

## 项目描述

通过特征工程和参数优化提升KNN模型的性能。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **特征工程**
   - 特征选择
   - 特征标准化
   - 特征变换

2. **参数优化**
   - K值选择
   - 距离度量选择
   - 权重方式选择

3. **性能提升**
   - 对比优化前后的性能
   - 分析优化效果
