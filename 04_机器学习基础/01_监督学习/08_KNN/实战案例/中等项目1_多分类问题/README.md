# 中等项目1：多分类问题

## 项目描述

使用KNN解决多分类问题，处理多个类别的分类任务。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据预处理**
   - 数据加载和清洗
   - 特征工程
   - 处理类别不平衡

2. **模型训练**
   - 训练KNN分类器
   - 参数调优
   - 交叉验证

3. **评估和分析**
   - 多分类性能指标
   - 可视化混淆矩阵
   - 分析各类别性能
