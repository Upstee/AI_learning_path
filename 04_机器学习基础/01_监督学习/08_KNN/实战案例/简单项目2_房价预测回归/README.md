# 简单项目2：房价预测回归

## 项目描述

使用KNN回归器预测房价，这是一个回归任务的示例。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备房价数据
   - 数据预处理
   - 特征选择

2. **模型训练**
   - 使用KNN回归器
   - 调整参数
   - 特征标准化

3. **评估和可视化**
   - 计算R²、MSE等指标
   - 可视化预测结果
   - 分析误差
