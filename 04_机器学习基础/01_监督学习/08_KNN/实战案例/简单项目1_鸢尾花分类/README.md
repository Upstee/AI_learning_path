# 简单项目1：鸢尾花分类

## 项目描述

使用KNN分类器对鸢尾花进行分类，这是机器学习中的经典入门项目。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 加载鸢尾花数据集
   - 数据探索和可视化
   - 划分训练集和测试集

2. **模型训练**
   - 使用KNN分类器
   - 调整K值
   - 特征标准化

3. **评估和可视化**
   - 计算准确率
   - 可视化决策边界
   - 分析结果
