# 中等项目1：文本分类系统

## 项目描述

使用SVM构建文本分类系统，进行情感分析或主题分类。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **文本预处理**
   - 文本清洗
   - 特征提取（TF-IDF等）
   - 数据预处理

2. **模型训练**
   - 训练SVM模型
   - 参数调优
   - 性能评估

3. **结果分析**
   - 分析分类性能
   - 识别关键特征词

## 项目文件

- `main.py` - 主程序
- `text_processing.py` - 文本处理
- `model_training.py` - 模型训练
- `README.md` - 项目说明（本文件）
