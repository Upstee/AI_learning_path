# 中等项目2：图像分类系统

## 项目描述

使用SVM进行图像分类，结合特征提取技术。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **特征提取**
   - 图像特征提取（HOG、SIFT等）
   - 特征降维
   - 数据预处理

2. **模型训练**
   - 训练SVM模型
   - 参数调优
   - 性能评估

3. **结果分析**
   - 分析分类性能
   - 可视化结果

## 项目文件

- `main.py` - 主程序
- `feature_extraction.py` - 特征提取
- `model_training.py` - 模型训练
- `README.md` - 项目说明（本文件）
