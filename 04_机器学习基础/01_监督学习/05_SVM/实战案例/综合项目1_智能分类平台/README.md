# 综合项目1：智能分类平台

## 项目描述

构建一个完整的智能分类平台，支持多种分类任务。

## 项目要求

### 难度等级
- **综合** - 整合多个知识点，解决综合问题
- **代码量**：1000-2000行
- **时间**：16-32小时

### 功能要求

1. **多任务支持**
   - 二分类任务
   - 多分类任务
   - 不同数据类型

2. **自动化流程**
   - 自动数据预处理
   - 自动模型选择
   - 自动参数调优

3. **用户界面**
   - Web界面或API
   - 数据上传
   - 结果展示

## 项目文件

- `core/` - 核心模块
- `tasks/` - 任务模块
- `ui/` - 用户界面
- `README.md` - 项目说明（本文件）
