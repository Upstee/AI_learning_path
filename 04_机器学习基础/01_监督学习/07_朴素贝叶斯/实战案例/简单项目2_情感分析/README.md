# 简单项目2：情感分析

## 项目描述

使用朴素贝叶斯分类器进行文本情感分析（正面/负面）。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备正面和负面情感文本数据
   - 文本预处理

2. **特征提取**
   - 提取文本特征
   - 使用TF-IDF或词频

3. **模型训练**
   - 训练朴素贝叶斯分类器
   - 评估性能

4. **应用**
   - 分析新文本的情感

- `main.py` - 主程序
- `README.md` - 项目说明（本文件）
