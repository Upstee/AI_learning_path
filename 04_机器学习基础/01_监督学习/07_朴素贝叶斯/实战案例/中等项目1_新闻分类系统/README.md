# 中等项目1：新闻分类系统

## 项目描述

构建一个新闻分类系统，能够自动将新闻分类到不同的类别（如体育、科技、娱乐等）。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据预处理**
   - 数据加载和清洗
   - 文本预处理（分词、去停用词等）
   - 特征工程（TF-IDF、N-gram等）

2. **模型训练**
   - 训练多种朴素贝叶斯模型
   - 参数调优
   - 模型评估

3. **系统集成**
   - 构建完整的分类系统
   - 提供预测接口

## 项目文件

- `main.py` - 主程序
- `preprocessing.py` - 数据预处理模块
- `model.py` - 模型训练模块
- `README.md` - 项目说明（本文件）
