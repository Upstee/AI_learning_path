# 简单项目1：垃圾邮件检测

## 项目描述

使用朴素贝叶斯分类器构建一个简单的垃圾邮件检测系统。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 准备垃圾邮件和正常邮件数据
   - 文本预处理（清洗、分词）

2. **特征提取**
   - 使用CountVectorizer提取词频特征
   - 或使用TfidfVectorizer提取TF-IDF特征

3. **模型训练**
   - 使用MultinomialNB或BernoulliNB
   - 训练模型

4. **评估和测试**
   - 计算准确率
   - 测试新邮件
