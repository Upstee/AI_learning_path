# 中等项目2：多分类问题

## 项目描述

使用朴素贝叶斯解决多分类问题，处理多个类别的分类任务。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **多分类处理**
   - 处理多个类别（3个以上）
   - 使用一对多或一对一策略（如果需要）

2. **特征工程**
   - 特征选择
   - 特征变换
   - 处理类别不平衡

3. **模型优化**
   - 参数调优
   - 性能评估
   - 结果分析
