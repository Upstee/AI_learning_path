# 中等项目2：医疗诊断辅助系统

## 项目描述

使用逻辑回归构建医疗诊断辅助系统，帮助医生进行疾病诊断。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据准备**
   - 加载医疗数据集（如糖尿病、心脏病等）
   - 数据质量检查
   - 数据探索

2. **特征工程**
   - 特征选择（医学特征）
   - 特征标准化
   - 处理缺失值

3. **模型训练**
   - 使用逻辑回归训练诊断模型
   - 超参数调优
   - 交叉验证

4. **模型评估**
   - 敏感性和特异性
   - ROC曲线
   - 混淆矩阵
   - 特征重要性（哪些症状/指标最重要）

5. **结果解释**
   - 模型可解释性
   - 诊断规则
   - 风险评估

## 实现要点

- 医学数据的特殊处理
- 高准确率和可靠性要求
- 模型可解释性
- 风险评估和阈值选择

## 预期结果

- 医疗诊断辅助系统
- 高准确率（> 0.80）
- 清晰的诊断规则
- 风险评估报告

## 注意事项

- 医疗数据需要谨慎处理
- 模型不能替代医生诊断
- 需要专业医学知识验证

## 项目文件

- `main.py` - 主程序
- `data_processing.py` - 数据处理
- `model.py` - 模型训练
- `evaluation.py` - 模型评估
- `README.md` - 项目说明（本文件）
