# 中等项目1：完整客户流失预测系统

## 项目描述

构建一个完整的客户流失预测系统，包括数据探索、特征工程、模型训练、评估和部署准备。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据探索（EDA）**
   - 数据基本信息
   - 缺失值分析
   - 特征分布分析
   - 类别不平衡分析
   - 相关性分析

2. **数据预处理**
   - 缺失值处理
   - 异常值处理
   - 特征编码
   - 特征标准化

3. **特征工程**
   - 特征选择
   - 特征创建
   - 特征变换

4. **模型训练与优化**
   - 基础模型训练
   - 超参数调优
   - 交叉验证
   - 处理类别不平衡

5. **模型评估**
   - 多种评估指标
   - ROC曲线和AUC
   - PR曲线
   - 混淆矩阵
   - 特征重要性

6. **结果分析**
   - 模型性能分析
   - 特征重要性分析
   - 业务建议

## 实现要点

- 完整的端到端流程
- 详细的EDA分析
- 专业的特征工程
- 模型优化和评估
- 清晰的文档和注释

## 预期结果

- 完整的客户流失预测系统
- 模型性能良好（AUC > 0.75）
- 详细的评估报告
- 业务建议

## 项目文件

- `main.py` - 主程序
- `eda.py` - 数据探索分析
- `preprocessing.py` - 数据预处理
- `modeling.py` - 模型训练
- `evaluation.py` - 模型评估
- `README.md` - 项目说明（本文件）
