# 简单项目1：鸢尾花分类

## 项目描述

使用决策树对鸢尾花进行分类，这是一个经典的分类任务。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据加载**
   - 加载鸢尾花数据集
   - 数据探索

2. **模型训练**
   - 训练决策树分类器
   - 评估模型性能

3. **结果分析**
   - 可视化决策树
   - 分析特征重要性
