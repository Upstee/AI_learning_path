# 中等项目1：客户流失预测系统

## 项目描述

构建一个完整的客户流失预测系统，使用决策树进行分类。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据预处理**
   - 数据加载和清洗
   - 特征工程
   - 数据标准化

2. **模型训练和调优**
   - 训练决策树模型
   - 参数调优
   - 交叉验证

3. **模型评估**
   - 性能评估
   - 特征重要性分析
   - 决策规则提取

4. **结果解释**
   - 可视化决策树
   - 解释关键决策规则

## 项目文件

- `main.py` - 主程序
- `data_preprocessing.py` - 数据预处理
- `model_training.py` - 模型训练
- `README.md` - 项目说明（本文件）
