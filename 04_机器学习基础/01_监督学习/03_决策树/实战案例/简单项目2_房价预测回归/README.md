# 简单项目2：房价预测回归

## 项目描述

使用决策树回归器预测房价，这是一个回归任务。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 加载房价数据
   - 数据预处理

2. **模型训练**
   - 训练决策树回归器
   - 评估回归性能

3. **结果分析**
   - 可视化预测结果
   - 分析特征重要性
