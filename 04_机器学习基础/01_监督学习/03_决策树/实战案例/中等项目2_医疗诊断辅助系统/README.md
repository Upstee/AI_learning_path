# 中等项目2：医疗诊断辅助系统

## 项目描述

使用决策树构建医疗诊断辅助系统，能够根据症状预测疾病。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据准备**
   - 医疗数据预处理
   - 特征工程
   - 类别不平衡处理

2. **模型开发**
   - 训练决策树模型
   - 多分类问题处理
   - 模型验证

3. **可解释性**
   - 决策规则提取
   - 症状重要性分析
   - 诊断路径可视化

## 项目文件

- `main.py` - 主程序
- `diagnosis_system.py` - 诊断系统
- `README.md` - 项目说明（本文件）
