# 简单项目2：房价预测回归

## 项目描述

使用随机森林回归器预测房价，对比单棵回归树。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 加载房价数据
   - 数据预处理

2. **模型训练**
   - 训练随机森林回归器
   - 训练单棵回归树（对比）
   - 评估回归性能

3. **结果分析**
   - 可视化预测结果
   - 分析特征重要性
   - 对比两种模型
