# 中等项目2：特征选择系统

## 项目描述

使用随机森林的特征重要性进行特征选择，构建特征选择系统。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **特征重要性分析**
   - 计算特征重要性
   - 可视化特征重要性
   - 识别关键特征

2. **特征选择**
   - 基于重要性选择特征
   - 递归特征消除
   - 评估选择效果

3. **性能对比**
   - 对比使用全部特征和选择特征
   - 分析性能变化

## 项目文件

- `main.py` - 主程序
- `feature_selector.py` - 特征选择
- `README.md` - 项目说明（本文件）
