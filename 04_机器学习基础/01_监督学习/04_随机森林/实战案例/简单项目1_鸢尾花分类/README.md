# 简单项目1：鸢尾花分类

## 项目描述

使用随机森林对鸢尾花进行分类，对比单棵决策树。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据加载**
   - 加载鸢尾花数据集
   - 数据探索

2. **模型训练**
   - 训练随机森林分类器
   - 训练单棵决策树（对比）
   - 评估模型性能

3. **结果分析**
   - 可视化特征重要性
   - 对比两种模型的性能
   - 分析OOB误差
