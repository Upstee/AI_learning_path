# 简单项目2：销售预测

## 项目描述

使用线性回归预测销售额，练习多特征回归。

## 项目要求

### 难度等级
- **简单** - 使用现成工具，多特征回归
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 特征：广告投入、促销活动、季节因子（3-4个特征）

2. **模型训练**
   - 使用LinearRegression
   - 多特征回归

3. **模型评估**
   - 计算评估指标
   - 分析特征重要性

4. **结果展示**
   - 打印评估指标和特征系数
   - 可视化
