# 简单项目1：基础房价预测

## 项目描述

使用线性回归预测房价，这是最简单的回归项目。

## 项目要求

### 难度等级
- **简单** - 使用现成工具，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 使用模拟数据或简单数据集
   - 特征：面积、房间数（2-3个特征即可）

2. **模型训练**
   - 使用scikit-learn的LinearRegression
   - 训练-测试集分割

3. **模型评估**
   - 计算MSE和R²
   - 简单可视化

4. **结果展示**
   - 打印评估指标
   - 绘制预测结果图

## 交付物

1. **代码文件**：`main.py`
2. **README**：项目说明和运行指南
3. **结果截图**：可视化结果

## 评估标准

- 代码能够运行
- 理解基本流程
- 结果合理
