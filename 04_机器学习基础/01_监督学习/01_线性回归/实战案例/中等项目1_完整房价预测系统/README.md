# 中等项目1：完整房价预测系统

## 项目描述

实现一个完整的房价预测系统，包括数据探索、预处理、模型训练、评估和特征分析。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据探索**
   - 统计信息、缺失值检查
   - 数据可视化（分布、相关性）
   - 异常值检测

2. **数据预处理**
   - 处理缺失值
   - 特征标准化
   - 特征工程（可选）

3. **模型训练和选择**
   - 训练多个模型（线性回归、Ridge、Lasso）
   - 使用交叉验证
   - 超参数调优

4. **模型评估**
   - 多个评估指标
   - 可视化（预测vs真实、残差图）
   - 特征重要性分析

5. **结果分析**
   - 模型性能分析
   - 特征重要性解释
   - 改进建议
