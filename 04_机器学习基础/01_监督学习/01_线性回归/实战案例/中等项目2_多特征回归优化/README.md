# 中等项目2：多特征回归优化

## 项目描述

使用多个特征进行回归，重点练习特征工程和模型优化。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **多特征数据**
   - 使用10+个特征
   - 包含数值特征和类别特征

2. **特征工程**
   - 处理类别特征（编码）
   - 特征选择
   - 特征交互（可选）

3. **模型优化**
   - 尝试多种正则化方法
   - 超参数调优
   - 模型选择

4. **性能提升**
   - 对比优化前后的性能
   - 分析改进效果

## 交付物

1. **代码文件**：完整的实现
2. **优化报告**：说明优化过程和效果
