# 简单项目1：集成方法对比

## 项目描述

对比不同的集成学习方法，理解各自的优势和适用场景。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **实现多种方法**
   - Bagging
   - AdaBoost
   - GBDT

2. **性能对比**
   - 对比准确率
   - 对比训练时间
   - 分析优缺点

3. **结果可视化**
   - 可视化性能对比
   - 生成对比报告

- `main.py` - 主程序
