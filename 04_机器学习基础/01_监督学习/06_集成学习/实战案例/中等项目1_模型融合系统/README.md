# 中等项目1：模型融合系统

## 项目描述

构建一个模型融合系统，使用Stacking等方法融合多个模型。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **多种基学习器**
   - 训练多个不同的基学习器
   - 评估每个基学习器的性能

2. **Stacking实现**
   - 实现Stacking方法
   - 使用交叉验证生成元特征
   - 训练元学习器

3. **融合效果分析**
   - 对比单个模型和融合模型
   - 分析融合效果

## 项目文件

- `main.py` - 主程序
- `stacking.py` - Stacking实现
- `README.md` - 项目说明（本文件）
