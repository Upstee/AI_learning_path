# 简单项目2：回归模型评估系统

## 项目描述

构建一个完整的回归模型评估系统，包括回归评估指标计算、交叉验证、模型比较和可视化等功能。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 加载或生成回归数据集
   - 数据预处理和划分

2. **模型训练**
   - 训练多个回归模型（线性回归、随机森林、梯度提升）
   - 进行预测

3. **模型评估**
   - 计算回归评估指标（MSE、RMSE、MAE、R²）
   - 可视化预测结果
   - 残差分析

4. **交叉验证**
   - 实现K折交叉验证
   - 比较不同模型的性能

5. **结果展示**
   - 生成评估报告
   - 可视化评估结果

## 技术栈

- **Python 3.8+**
- **scikit-learn** - 模型和评估工具
- **matplotlib** - 数据可视化
- **pandas** - 数据处理
- **numpy** - 数值计算

## 运行指南

运行完成后，你将看到：
