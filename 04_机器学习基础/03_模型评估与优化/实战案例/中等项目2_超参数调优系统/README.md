# 中等项目2：超参数调优系统

## 项目描述

构建一个完整的超参数调优系统，包括网格搜索、随机搜索、贝叶斯优化等方法，支持多种评估指标和交叉验证策略。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据准备**
   - 加载分类或回归数据集
   - 数据预处理
   - 数据划分

2. **超参数搜索方法**
   - 网格搜索（Grid Search）
   - 随机搜索（Random Search）
   - 贝叶斯优化（Bayesian Optimization，使用optuna）

3. **模型支持**
   - 支持多种模型（随机森林、SVM、梯度提升等）
   - 可配置的参数空间

4. **评估策略**
   - K折交叉验证
   - 分层交叉验证
   - 多种评估指标

5. **结果分析**
   - 最佳参数展示
   - 参数重要性分析
   - 参数搜索过程可视化
   - 性能对比

6. **结果保存**
   - 保存最佳模型
   - 保存搜索结果
   - 生成调优报告

## 技术栈

- **Python 3.8+**
- **scikit-learn** - 模型和搜索工具
- **optuna** - 贝叶斯优化框架
- **matplotlib** - 数据可视化
- **seaborn** - 高级数据可视化
- **pandas** - 数据处理
- **numpy** - 数值计算

运行完成后，你将看到：

4. **搜索过程可视化**：参数搜索的优化过程
