# 简单项目1：模型评估系统

## 项目描述

构建一个完整的模型评估系统，包括评估指标计算、交叉验证、模型比较等功能。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 300行
- **时间**：3-5小时

### 功能要求

1. **数据准备**
   - 准备分类和回归数据
   - 数据预处理

2. **模型评估**
   - 计算分类评估指标（准确率、精确率、召回率、F1、ROC、AUC）
   - 计算回归评估指标（MSE、RMSE、MAE、R²）
   - 可视化评估结果

3. **交叉验证**
   - 实现K折交叉验证
   - 实现分层交叉验证
   - 比较不同模型

4. **结果展示**
   - 生成评估报告
   - 可视化评估结果
