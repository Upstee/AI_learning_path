# 中等项目1：多分类模型评估系统

## 项目描述

构建一个完整的多分类模型评估系统，包括多分类评估指标计算、混淆矩阵分析、分类报告生成、模型比较和可视化等功能。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据准备**
   - 加载多分类数据集（如Iris、Wine等）
   - 数据预处理和特征工程
   - 数据划分

2. **模型训练**
   - 训练多个多分类模型（逻辑回归、决策树、随机森林、SVM等）
   - 支持一对多（OvR）和一对一（OvO）策略

3. **模型评估**
   - 计算多分类评估指标（准确率、精确率、召回率、F1、宏平均、微平均、加权平均）
   - 生成混淆矩阵
   - 生成分类报告
   - 支持多类别ROC曲线（一对多）

4. **可视化**
   - 混淆矩阵热力图
   - 评估指标对比图
   - 多类别ROC曲线
   - 特征重要性分析

5. **交叉验证**
   - K折交叉验证
   - 分层交叉验证
   - 模型性能比较

6. **结果分析**
   - 生成详细评估报告
   - 模型性能排名
   - 错误分析

## 技术栈

- **Python 3.8+**
- **scikit-learn** - 模型和评估工具
- **matplotlib** - 数据可视化
- **seaborn** - 高级数据可视化
- **pandas** - 数据处理
- **numpy** - 数值计算

## 结果展示

运行完成后，你将看到：

1. **评估指标表格**：各模型的详细评估指标（宏平均、微平均、加权平均）
2. **混淆矩阵热力图**：每个模型的混淆矩阵可视化
3. **多类别ROC曲线**：一对多策略的ROC曲线
4. **模型性能对比**：多个模型的综合性能对比
5. **交叉验证结果**：各模型的交叉验证得分和标准差
6. **详细评估报告**：包含每个类别的精确率、召回率、F1分数
