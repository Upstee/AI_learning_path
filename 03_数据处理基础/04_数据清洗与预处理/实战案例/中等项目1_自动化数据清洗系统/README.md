# 中等项目1：自动化数据清洗系统

## 项目描述

构建一个自动化数据清洗系统，能够自动检测和处理数据质量问题。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **自动检测**
   - 自动检测缺失值
   - 自动检测异常值
   - 自动检测数据类型

2. **自动处理**
   - 根据数据特点自动选择处理方法
   - 自动标准化
   - 自动特征工程

3. **处理报告**
   - 生成处理报告
   - 记录处理步骤

## 项目文件

- `main.py` - 主程序
- `auto_cleaner.py` - 自动清洗类
- `README.md` - 项目说明（本文件）
