# 进阶项目2：智能特征工程系统

## 项目描述

构建一个智能特征工程系统，能够自动创建和选择特征。

## 项目要求

### 难度等级
- **进阶** - 需要优化和调参，处理实际问题
- **代码量**：500-1000行
- **时间**：8-16小时

### 功能要求

1. **自动特征创建**
   - 自动特征组合
   - 自动特征变换
   - 自动特征选择

2. **特征评估**
   - 特征重要性评估
   - 特征贡献分析

3. **性能优化**
   - 高效的特征计算
   - 特征缓存机制

## 项目文件

- `main.py` - 主程序
- `feature_creator.py` - 特征创建
- `feature_selector.py` - 特征选择
- `README.md` - 项目说明（本文件）
