# 中等项目2：特征工程工具库

## 项目描述

构建一个特征工程工具库，包含常用的特征工程方法。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **特征创建**
   - 特征组合
   - 特征变换
   - 时间特征提取

2. **特征选择**
   - 过滤法
   - 包装法
   - 嵌入法

3. **特征编码**
   - 独热编码
   - 标签编码
   - 目标编码

## 项目文件

- `main.py` - 主程序
- `feature_engineering.py` - 特征工程类
- `README.md` - 项目说明（本文件）
