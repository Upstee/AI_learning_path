# 简单项目2：数据标准化工具

## 项目描述

使用sklearn构建一个数据标准化工具，支持多种标准化方法。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **多种标准化方法**
   - Z-score标准化
   - Min-Max归一化
   - Robust标准化

2. **特征工程**
   - 特征变换
   - 特征编码

3. **结果验证**
   - 验证标准化效果
   - 保存处理后的数据

- `main.py` - 主程序
