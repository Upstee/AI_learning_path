# 简单项目1：数据清洗工具

## 项目描述

使用Pandas构建一个数据清洗工具，能够处理常见的质量问题。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **缺失值处理**
   - 检测缺失值
   - 删除或填充缺失值

2. **异常值处理**
   - 检测异常值
   - 处理异常值

3. **数据去重**
   - 检测重复值
   - 删除重复值

- `main.py` - 主程序
