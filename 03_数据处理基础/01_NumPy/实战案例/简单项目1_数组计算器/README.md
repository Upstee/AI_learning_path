# 简单项目1：数组计算器

## 项目描述

使用NumPy构建一个数组计算器，能够进行基本的数组运算和统计。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数组创建**
   - 从用户输入创建数组
   - 创建特殊数组（全零、全1、随机等）

2. **数组运算**
   - 基本运算（加减乘除）
   - 统计运算（平均值、标准差等）
   - 数组变换（reshape、转置等）

3. **结果展示**
   - 清晰的输出格式
   - 基本的错误处理

## 实现要点

- 使用NumPy完成所有计算
- 用户友好的界面
- 基本的输入验证

## 预期结果

- 功能完整的数组计算器
- 能够处理常见数组操作
- 清晰的输出
