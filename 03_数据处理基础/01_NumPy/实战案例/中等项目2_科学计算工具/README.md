# 中等项目2：科学计算工具

## 项目描述

使用NumPy构建一个科学计算工具，包含矩阵运算、线性代数、数值积分等功能。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **矩阵运算**
   - 矩阵乘法
   - 矩阵求逆
   - 行列式计算
   - 特征值分解

2. **线性方程组求解**
   - 求解线性方程组
   - 最小二乘解
   - 条件数分析

3. **数值计算**
   - 数值积分（梯形法、辛普森法）
   - 数值微分
   - 插值计算

4. **统计分析**
   - 相关性分析
   - 协方差矩阵
   - 主成分分析（基础实现）

## 实现要点

- 使用NumPy的线性代数模块
- 实现数值算法
- 代码模块化
- 包含测试和验证

## 预期结果

- 功能完整的科学计算工具
- 准确的数值计算
- 清晰的代码结构

## 项目文件

- `main.py` - 主程序
- `matrix_ops.py` - 矩阵运算
- `numerical_methods.py` - 数值方法
- `README.md` - 项目说明（本文件）
