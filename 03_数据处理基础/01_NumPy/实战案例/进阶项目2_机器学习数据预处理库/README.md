# 进阶项目2：机器学习数据预处理库

## 项目描述

使用NumPy构建一个机器学习数据预处理库，包含标准化、归一化、特征工程等功能。

## 项目要求

### 难度等级
- **进阶** - 需要优化和调参，处理实际问题
- **代码量**：500-1000行
- **时间**：8-16小时

### 功能要求

1. **数据标准化**
   - Z-score标准化
   - Min-Max归一化
   - Robust标准化

2. **特征工程**
   - 特征选择
   - 特征变换（对数、平方根等）
   - 特征组合

3. **数据清洗**
   - 缺失值处理
   - 异常值检测和处理
   - 数据验证

4. **数据分割**
   - 训练集/测试集分割
   - K折交叉验证数据分割

5. **批处理支持**
   - 批量数据处理
   - 数据管道

## 实现要点

- 完整的预处理流程
- 向量化实现
- 可扩展的架构
- 完整的测试

## 预期结果

- 功能完整的预处理库
- 高效的实现
- 清晰的API设计

## 项目文件

- `preprocessor.py` - 预处理类
- `normalizers.py` - 标准化器
- `feature_engineering.py` - 特征工程
- `data_cleaning.py` - 数据清洗
- `README.md` - 项目说明（本文件）
