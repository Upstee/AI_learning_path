# 中等项目1：图像处理工具

## 项目描述

使用NumPy构建一个图像处理工具，能够进行基本的图像操作和变换。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **图像加载和保存**
   - 加载图像文件（使用PIL/Pillow）
   - 转换为NumPy数组
   - 保存处理后的图像

2. **图像变换**
   - 亮度调整
   - 对比度调整
   - 图像翻转
   - 图像旋转
   - 图像裁剪

3. **图像滤波**
   - 均值滤波
   - 高斯滤波（使用NumPy实现）

4. **图像统计**
   - 直方图计算
   - 图像质量评估

## 实现要点

- 使用NumPy进行所有图像处理
- 向量化操作，避免循环
- 清晰的代码结构
- 基本的可视化

## 预期结果

- 功能完整的图像处理工具
- 能够处理常见图像操作
- 清晰的代码和文档

- `main.py` - 主程序
