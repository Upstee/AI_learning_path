# 简单项目2：数据统计分析工具

## 项目描述

使用NumPy构建一个数据统计分析工具，能够加载数据并进行统计分析。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据加载**
   - 从文件或手动输入加载数据
   - 数据验证和清洗

2. **统计分析**
   - 描述性统计（均值、中位数、标准差等）
   - 分位数计算
   - 数据分布分析

3. **结果展示**
   - 清晰的统计报告
   - 基本可视化（可选）

## 实现要点

- 使用NumPy进行所有计算
- 处理不同类型的数据
- 清晰的输出格式

## 预期结果

- 功能完整的统计分析工具
- 能够生成统计报告
- 清晰的输出
