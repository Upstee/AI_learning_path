# 基础练习2：广播机制应用

## 练习目标

理解并应用NumPy的广播机制。

## 练习要求

### 1. 标量与数组的广播（20分）

给定数组：
```python
arr = np.array([[1, 2, 3],
                [4, 5, 6]])
```

完成以下操作（使用广播）：

1. 数组每个元素加10
2. 数组每个元素乘以2
3. 数组每个元素平方

### 2. 不同形状数组的广播（30分）

给定数组：
```python
arr1 = np.array([[1, 2, 3],
                 [4, 5, 6]])  # shape: (2, 3)
arr2 = np.array([10, 20, 30])  # shape: (3,)
```

完成以下操作：

1. 计算arr1 + arr2（解释广播过程）
2. 计算arr1 * arr2
3. 创建一个形状为(2, 1)的数组，与arr1进行广播运算

### 3. 广播的实际应用（30分）

1. **标准化数据**：
   - 给定一个2D数组，每行减去该行的平均值
   - 使用广播实现，不使用循环

2. **计算距离矩阵**：
   - 给定两个点集（每个点有x, y坐标）
   - 计算所有点对之间的欧氏距离
   - 使用广播避免循环

3. **外积计算**：
   - 给定两个1D数组
   - 使用广播计算外积（不使用np.outer）

### 4. 广播规则验证（20分）

判断以下数组对是否可以广播，如果可以，说明结果形状：

1. `(3, 4)` 和 `(4,)`
2. `(3, 4)` 和 `(3, 1)`
3. `(3, 4)` 和 `(1, 4)`
4. `(3, 4)` 和 `(3, 4)`
5. `(3, 4)` 和 `(2, 3, 4)`

## 代码要求

- 使用NumPy广播机制
- 不使用Python循环
- 代码清晰，有注释
- 解释广播过程

## 评估标准

- **正确性**（60分）：所有操作正确
- **理解深度**（20分）：能够解释广播过程
- **代码质量**（20分）：代码清晰，使用广播而非循环

## 提示

- 广播规则：从右到左比较维度
- 维度为1或相同可以广播
- 使用`np.newaxis`或`None`增加维度
