# 进阶练习1：图像处理基础

## 练习目标

使用NumPy进行图像处理的基础操作。

## 练习要求

### 1. 图像表示（20分）

1. **创建模拟图像**：
   - 创建一个256x256的灰度图像（值在0-255之间）
   - 创建一个256x256x3的RGB图像

2. **图像属性**：
   - 获取图像的形状、数据类型、大小
   - 计算图像占用的内存

### 2. 图像操作（30分）

给定一个图像数组（可以使用`np.random.randint(0, 256, (256, 256))`模拟）：

1. **图像裁剪**：
   - 裁剪图像的中心100x100区域
   - 裁剪图像的左上角50x50区域

2. **图像翻转**：
   - 水平翻转图像
   - 垂直翻转图像

3. **图像旋转**：
   - 使用转置实现90度旋转
   - 实现180度旋转

### 3. 图像变换（30分）

1. **亮度调整**：
   - 将图像亮度增加50（使用广播）
   - 将图像亮度减少30
   - 确保值在0-255范围内（使用`np.clip`）

2. **对比度调整**：
   - 将图像对比度增加（乘以1.5）
   - 将图像对比度减少（乘以0.7）

3. **颜色通道操作**（RGB图像）：
   - 提取红色通道
   - 提取绿色通道
   - 提取蓝色通道
   - 交换红色和蓝色通道

### 4. 图像统计（20分）

1. 计算图像的平均亮度
2. 计算每个颜色通道的平均值（RGB图像）
3. 找出图像中最亮的像素位置
4. 找出图像中最暗的像素位置
5. 计算图像的直方图（使用`np.histogram`）

## 代码要求

- 使用NumPy完成所有操作
- 不使用Python循环
- 代码模块化（使用函数）
- 包含可视化（使用matplotlib显示图像）

## 评估标准

- **正确性**（50分）：所有操作正确
- **代码质量**（30分）：代码清晰，模块化
- **可视化**（20分）：图像显示清晰

## 提示

- 图像通常用uint8类型（0-255）
- 使用`np.clip()`限制值范围
- 使用`matplotlib.pyplot.imshow()`显示图像
- RGB图像形状为(height, width, 3)
