# 简单项目2：统计图表生成器

## 项目描述

使用Seaborn构建一个统计图表生成器，能够快速创建统计图表。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **统计图表**
   - 分布图
   - 箱线图
   - 小提琴图
   - 回归图

2. **数据输入**
   - 从DataFrame读取
   - 支持分组数据

3. **图表保存**
   - 保存为高质量图片

- `main.py` - 主程序
