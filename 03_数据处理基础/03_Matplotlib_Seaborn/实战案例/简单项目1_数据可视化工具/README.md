# 简单项目1：数据可视化工具

## 项目描述

使用Matplotlib和Seaborn构建一个数据可视化工具，能够快速创建常见图表。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **图表创建**
   - 支持多种图表类型
   - 简单的参数配置

2. **数据输入**
   - 从Pandas DataFrame读取
   - 从CSV文件读取

3. **图表保存**
   - 保存为图片文件
   - 支持不同格式

## 项目文件

- `main.py` - 主程序
- `visualizer.py` - 可视化类
- `README.md` - 项目说明（本文件）
