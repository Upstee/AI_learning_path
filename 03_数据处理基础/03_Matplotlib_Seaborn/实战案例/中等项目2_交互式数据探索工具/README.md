# 中等项目2：交互式数据探索工具

## 项目描述

使用交互式可视化库（如Plotly）构建数据探索工具。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **交互式图表**
   - 交互式散点图
   - 交互式线图
   - 交互式热力图

2. **数据探索**
   - 数据筛选
   - 动态更新图表
   - 数据钻取

3. **界面设计**
   - 用户友好的界面
   - 清晰的布局

## 项目文件

- `main.py` - 主程序
- `interactive_plots.py` - 交互式图表
- `README.md` - 项目说明（本文件）
