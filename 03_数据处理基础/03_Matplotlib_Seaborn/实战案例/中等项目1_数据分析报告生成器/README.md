# 中等项目1：数据分析报告生成器

## 项目描述

构建一个数据分析报告生成器，自动生成包含多个图表的分析报告。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据加载**
   - 支持多种数据格式
   - 数据验证

2. **自动分析**
   - 自动生成多个图表
   - 数据分布分析
   - 关系分析

3. **报告生成**
   - 组合多个图表
   - 添加分析说明
   - 导出为PDF或HTML

## 项目文件

- `main.py` - 主程序
- `report_generator.py` - 报告生成类
- `README.md` - 项目说明（本文件）
