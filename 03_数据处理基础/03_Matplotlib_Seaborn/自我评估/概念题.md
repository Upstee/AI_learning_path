## 二、简答题（每题10分，共40分）

### 1. 解释Matplotlib的三层架构。

**参考答案**：
- **Backend层**：处理图形渲染
- **Artist层**：图形对象（Figure, Axes等）
- **Scripting层**：pyplot接口（面向用户的API）

---

### 2. 说明如何选择合适的图表类型。

**参考答案**：
- 连续数据：线图、散点图、直方图
- 分类数据：柱状图、饼图、箱线图
- 时间序列：线图
- 关系分析：散点图、热力图

---

### 3. 解释Seaborn的统计可视化优势。

**参考答案**：
- 内置统计功能（如回归图、分布图）
- 美观的默认样式
- 多变量可视化支持
- 与Pandas集成

---

### 4. 说明数据可视化的设计原则。

**参考答案**：
- **清晰性**：图表清晰易懂
- **准确性**：准确反映数据
- **美观性**：视觉上吸引人
- **一致性**：统一的风格

## 三、计算题（每题10分，共20分）

### 1. 给定数据，创建以下图表：
- 线图：展示y = sin(x)
- 散点图：展示x和y的关系
- 柱状图：展示分类数据

**参考答案**：
```python
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

# 线图
plt.plot(x, y)
plt.title('sin(x)')

# 散点图
plt.scatter(x, y)

# 柱状图
categories = ['A', 'B', 'C']
values = [10, 20, 30]
plt.bar(categories, values)
```
