## 练习要求

### 1. 分组聚合（40分）

给定数据：
```python
df = pd.DataFrame({
    'category': ['A', 'A', 'B', 'B', 'A', 'B', 'C', 'C'],
    'value': [10, 20, 30, 40, 50, 60, 70, 80],
    'count': [1, 2, 3, 4, 5, 6, 7, 8]
})
```

完成以下操作：

1. **基本分组**
   - 按category分组
   - 计算每组的平均值

2. **多列分组**
   - 按多个列分组
   - 计算多个统计量

3. **自定义聚合函数**
   - 使用agg()函数

### 2. 数据合并（40分）

创建两个DataFrame：
```python
df1 = pd.DataFrame({
    'key': ['A', 'B', 'C', 'D'],
    'value1': [1, 2, 3, 4]
})
df2 = pd.DataFrame({
    'key': ['B', 'C', 'D', 'E'],
    'value2': [5, 6, 7, 8]
})
```

完成以下操作：

1. **不同类型的合并**
   - 内连接（inner）
   - 外连接（outer）
   - 左连接（left）
   - 右连接（right）

2. **多键合并**
   - 基于多个键合并
   - 处理键名不同的情况

3. **拼接操作**
   - 使用concat()垂直拼接
   - 使用concat()水平拼接

### 3. 数据透视（20分）

1. **透视表**
   - 使用pivot_table()创建透视表
   - 指定行、列、值
   - 指定聚合函数

2. **交叉表**
   - 使用crosstab()创建交叉表
