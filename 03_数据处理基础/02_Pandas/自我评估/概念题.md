## 二、简答题（每题10分，共40分）

### 1. 解释Series和DataFrame的区别和联系。

**参考答案**：
- **Series**：一维带标签数组，类似带索引的列表
- **DataFrame**：二维带标签数据结构，可以看作多个Series的集合
- **联系**：DataFrame的每一列是一个Series，它们共享行索引

---

### 2. 说明iloc和loc的区别和使用场景。

**参考答案**：
- **iloc**：基于位置的索引，类似NumPy数组索引，速度快
- **loc**：基于标签的索引，支持标签对齐，更灵活
- **使用场景**：iloc用于位置访问，loc用于标签访问

---

### 3. 解释Pandas的数据对齐机制。

**参考答案**：
- 不同Series/DataFrame之间的运算会自动对齐索引
- 相同标签对应位置运算，不同标签结果包含所有标签，缺失值为NaN
- 避免位置错位导致的错误

---

### 4. 说明如何优化Pandas代码性能。

**参考答案**：
1. 使用向量化操作，避免Python循环
2. 选择合适的数据类型
3. 使用分类类型（category）
4. 使用适当的数据结构（如索引）
5. 分块处理大文件

## 三、计算题（每题10分，共20分）

### 1. 给定DataFrame，完成以下操作：
```python
df = pd.DataFrame({
    'A': [1, 2, 3, 4, 5],
    'B': [10, 20, 30, 40, 50],
    'C': ['a', 'b', 'c', 'd', 'e']
})
```

计算：
- df['A'] + df['B']
- df.groupby('C')['A'].sum()
- df[df['A'] > 2]

**参考答案**：
- df['A'] + df['B']: [11, 22, 33, 44, 55]
- df.groupby('C')['A'].sum(): 每个组的A列求和
- df[df['A'] > 2]: 筛选A列大于2的行

---

### 2. 给定两个DataFrame，完成合并操作：
```python
df1 = pd.DataFrame({'key': ['A', 'B'], 'value1': [1, 2]})
df2 = pd.DataFrame({'key': ['B', 'C'], 'value2': [3, 4]})
```

计算内连接、外连接、左连接的结果。

**参考答案**：
- 内连接：只保留key='B'的行
- 外连接：保留所有key（A, B, C），缺失值用NaN
- 左连接：保留df1的所有key（A, B），缺失值用NaN
