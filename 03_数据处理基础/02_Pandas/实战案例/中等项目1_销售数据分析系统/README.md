# 中等项目1：销售数据分析系统

## 项目描述

构建一个完整的销售数据分析系统，包括数据加载、清洗、分析和可视化。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据加载和探索**
   - 加载销售数据
   - 数据探索性分析
   - 数据质量检查

2. **数据清洗**
   - 处理缺失值
   - 处理异常值
   - 数据类型转换

3. **数据分析**
   - 按产品/地区/时间分组分析
   - 计算销售指标（总额、平均值、增长率等）
   - 趋势分析

4. **数据可视化**
   - 销售趋势图
   - 分组对比图
   - 热力图

5. **报告生成**
   - 生成分析报告
   - 保存处理后的数据

## 项目文件

- `main.py` - 主程序
- `data_analyzer.py` - 数据分析类
- `visualizer.py` - 可视化模块
- `README.md` - 项目说明（本文件）
