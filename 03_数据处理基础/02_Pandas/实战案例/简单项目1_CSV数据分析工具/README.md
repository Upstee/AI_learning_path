# 简单项目1：CSV数据分析工具

## 项目描述

使用Pandas构建一个CSV数据分析工具，能够加载、分析和可视化CSV数据。

## 项目要求

### 难度等级
- **简单** - 使用现成工具/库，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据加载**
   - 读取CSV文件
   - 处理编码问题
   - 数据预览

2. **数据分析**
   - 描述性统计
   - 数据分布分析
   - 缺失值分析

3. **结果输出**
   - 生成统计报告
   - 保存处理后的数据

- `main.py` - 主程序
