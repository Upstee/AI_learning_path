# 中等项目2：客户行为分析系统

## 项目描述

使用Pandas分析客户行为数据，包括购买记录、浏览历史等。

## 项目要求

### 难度等级
- **中等** - 需要理解原理，组合多个技术
- **代码量**：200-500行
- **时间**：4-8小时

### 功能要求

1. **数据整合**
   - 合并多个数据源
   - 数据对齐和清洗

2. **行为分析**
   - 客户分组（RFM分析）
   - 购买模式分析
   - 客户生命周期分析

3. **统计分析**
   - 客户留存率
   - 复购率
   - 平均订单价值

4. **结果输出**
   - 客户分群报告
   - 行为分析报告

## 项目文件

- `main.py` - 主程序
- `customer_analyzer.py` - 客户分析类
- `README.md` - 项目说明（本文件）
