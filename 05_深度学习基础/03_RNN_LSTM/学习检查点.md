## 🔍 学习中检查：我理解了吗？

### 阶段1：基本概念

- [ ] **RNN是什么？**
  - 我能解释RNN的工作原理
  - 我理解RNN的循环结构
  - 我知道RNN的梯度问题

- [ ] **LSTM是什么？**
  - 我理解LSTM的门控机制
  - 我知道LSTM如何解决梯度消失
  - 我理解遗忘门、输入门、输出门

### 阶段2：算法细节

- [ ] **RNN的数学原理？**
  - 我能写出RNN的前向传播公式
  - 我理解反向传播通过时间（BPTT）

- [ ] **LSTM的结构？**
  - 我能画出LSTM的结构图
  - 我理解每个门的作用

### 阶段3：代码实现

- [ ] **从零实现RNN/LSTM？**
  - 我能用框架实现RNN/LSTM
  - 我理解每个参数的含义

## ✅ 学习后评估：我掌握了吗？

### 概念掌握评估

**自测题**：
1. RNN的梯度消失问题是什么？
2. LSTM如何解决梯度消失？
3. 遗忘门、输入门、输出门的作用是什么？
4. RNN/LSTM在哪些场景中应用？

**评分标准**：
- 4题全对：✅ 概念掌握优秀
- 3题正确：✅ 概念掌握良好
- 2题正确：⚠️ 需要复习
- 少于2题：❌ 需要重新学习

## 🆘 如果卡住了：怎么办？

### 问题1：不理解循环结构

**解决方法**：
1. 展开RNN，理解时间步
2. 用简单例子理解（3个时间步）
3. 可视化RNN的展开

### 问题2：不理解LSTM的门控机制

**解决方法**：
1. 逐个理解每个门
2. 用流程图理解
3. 查看可视化工具
