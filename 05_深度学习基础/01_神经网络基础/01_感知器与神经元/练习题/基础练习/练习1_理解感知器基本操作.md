## 练习要求

### 任务1：手动计算感知器输出

给定：
- 权重：$\mathbf{w} = [0.5, -0.3]$
- 偏置：$b = 0.2$
- 输入：$\mathbf{x} = [1, 2]$

**要求**：
1. 计算加权和（净输入）：$z = \mathbf{w}^T \mathbf{x} + b$
2. 通过阶跃函数得到输出：$y = f(z)$
3. 解释每一步的含义

**参考答案**：
```python
import numpy as np

w = np.array([0.5, -0.3])
b = 0.2
x = np.array([1, 2])

# 计算加权和
z = np.dot(w, x) + b  # z = 0.5*1 + (-0.3)*2 + 0.2 = 0.1

# 通过阶跃函数
y = 1 if z >= 0 else 0  # y = 1

print(f"加权和 z = {z:.4f}")
print(f"输出 y = {y}")
```

**解释**：
- 加权和 $z = 0.1 \geq 0$，所以输出 $y = 1$
- 这意味着该输入被分类为正类

### 任务3：理解决策边界

**要求**：

2. 判断点 $(1, 2)$ 和 $(-1, 1)$ 在决策边界的哪一侧

**提示**：
