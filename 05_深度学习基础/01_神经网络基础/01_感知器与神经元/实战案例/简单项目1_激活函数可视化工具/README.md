# 简单项目1：激活函数可视化工具

## 项目目标

创建一个激活函数可视化工具，帮助理解不同激活函数的特性。

## 项目概述

本项目将：
1. 实现常用的激活函数（Sigmoid、Tanh、ReLU等）
2. 可视化激活函数及其导数
3. 对比不同激活函数的特性

## 项目要求

- **难度**：简单
- **代码量**：< 200行
- **预计时间**：2-4小时
- **功能要求**：
  - 实现至少3种激活函数
  - 绘制激活函数图像
  - 绘制导数图像
  - 添加说明文字

## 实现步骤

1. **实现激活函数类**
   - Sigmoid函数
   - Tanh函数
   - ReLU函数

2. **可视化功能**
   - 绘制激活函数曲线
   - 绘制导数曲线
   - 添加图例和标签

3. **对比分析**
   - 输出范围对比
   - 梯度特性对比
