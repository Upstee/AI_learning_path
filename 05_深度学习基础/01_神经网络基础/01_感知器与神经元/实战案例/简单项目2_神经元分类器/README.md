# 简单项目2：神经元分类器

## 项目目标

使用单个神经元（带Sigmoid激活函数）实现一个二分类器，类似于逻辑回归。

## 项目概述

本项目将：
1. 实现一个带Sigmoid激活函数的神经元
2. 使用梯度下降训练神经元
3. 在二分类数据集上测试性能

## 项目要求

- **难度**：简单
- **代码量**：< 200行
- **预计时间**：2-4小时
- **功能要求**：
  - 实现神经元类（前向传播、反向传播）
  - 实现训练循环
  - 计算准确率
  - 可视化决策边界

## 实现步骤

1. **实现神经元类**
   - 初始化权重和偏置
   - 前向传播（计算输出）
   - 反向传播（计算梯度）
   - 参数更新

2. **训练过程**
   - 定义损失函数（均方误差或交叉熵）
   - 实现训练循环
   - 记录训练损失

3. **评估和可视化**
   - 计算准确率
   - 绘制决策边界
   - 可视化训练过程

## 数据集

使用简单的线性可分数据集（如Iris数据集的前两个特征，或自己生成的数据）。

## 参考代码

参考 `代码示例/01_从零实现神经元.ipynb` 中的Neuron类实现。
