# 简单项目1：感知器分类器

## 项目描述

使用感知器实现一个简单的二分类器，解决线性可分的分类问题。

## 项目要求

### 难度等级
- **简单** - 使用感知器，完成基础任务
- **代码量**：< 200行
- **时间**：2-4小时

### 功能要求

1. **数据准备**
   - 生成或加载线性可分的二分类数据
   - 数据可视化

2. **感知器实现**
   - 从零实现感知器类
   - 实现训练和预测功能

3. **模型训练**
   - 训练感知器
   - 可视化训练过程

4. **结果评估**
   - 计算准确率
   - 可视化决策边界
   - 分析分类结果

## 技术栈

- **Python 3.8+**
- **NumPy** - 数值计算
- **Matplotlib** - 数据可视化

## 运行指南

### 环境准备

```bash
pip install numpy matplotlib
```

### 运行方式

1. **使用Jupyter Notebook**（推荐）：
   ```bash
   jupyter notebook main.ipynb
   ```

## 结果展示

运行完成后，你将看到：

1. **训练数据分布**：两类数据的散点图
2. **训练过程**：错误数随迭代次数的变化
3. **决策边界**：感知器学习到的分类边界
4. **分类准确率**：模型在训练集上的表现
