# 简单项目2：非线性分类

## 项目概述

使用前馈神经网络解决非线性分类问题（月牙形数据集），展示多层网络处理复杂模式的能力。

## 项目目标

1. 理解前馈神经网络处理非线性问题的能力
2. 比较不同网络结构的效果
3. 可视化决策边界
4. 分析网络深度和宽度的影响

## 数据集

使用sklearn的`make_moons`生成月牙形数据集：
- 样本数：1000
- 噪声：0.1
- 二分类问题

## 实现步骤

### 1. 数据准备

```python
from sklearn.datasets import make_moons
X, y = make_moons(n_samples=1000, noise=0.1, random_state=42)
```

### 2. 网络结构对比

**实验1：浅层网络**
- 输入层：2
- 隐藏层：8（ReLU）
- 输出层：1（Sigmoid）

**实验2：深层网络**
- 输入层：2
- 隐藏层1：16（ReLU）
- 隐藏层2：8（ReLU）
- 输出层：1（Sigmoid）

**实验3：宽网络**
- 输入层：2
- 隐藏层：32（ReLU）
- 输出层：1（Sigmoid）

### 3. 训练和评估

- 划分训练集和验证集（80/20）
- 训练1000轮
- 记录训练和验证损失、准确率
- 绘制训练曲线

### 4. 可视化

- 绘制决策边界
- 比较不同网络结构的决策边界
- 分析过拟合现象

## 预期结果

- 所有网络都能学习非线性决策边界
- 深层网络可能更容易过拟合
- 宽网络需要更多参数

## 扩展练习

1. 尝试不同的激活函数组合
2. 添加正则化（Dropout、L2）防止过拟合
3. 使用早停（Early Stopping）
4. 分析隐藏层的激活值分布
