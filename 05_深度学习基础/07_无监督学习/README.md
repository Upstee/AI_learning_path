## 3. 核心知识点详解

### 3.1 无监督特征学习

#### 3.1.1 自编码器（Autoencoder）

**结构**：
```
输入 → 编码器 → 潜在表示 → 解码器 → 重构输出
```

**目标**：学习数据的紧凑表示，使得重构误差最小。

**应用**：
- 降维
- 特征学习
- 去噪

#### 3.1.2 变分自编码器（VAE）

**特点**：学习潜在空间的概率分布。

**优势**：可以生成新样本。

### 3.2 生成模型

#### 3.2.1 生成对抗网络（GAN）

**结构**：
- **生成器（Generator）**：生成假样本
- **判别器（Discriminator）**：区分真假样本

**训练过程**：生成器和判别器对抗训练。

### 3.3 分离表示学习

**目标**：学习数据的分离表示，每个维度对应不同的语义。
