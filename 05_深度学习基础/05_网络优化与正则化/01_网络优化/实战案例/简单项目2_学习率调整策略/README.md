# 简单项目2：学习率调整策略

## 项目目标

实现和对比不同的学习率调整策略，理解学习率对训练的影响。

## 项目概述

本项目将：
1. 实现多种学习率调整策略（固定、衰减、余弦退火等）
2. 在相同模型和数据集上测试不同策略
3. 可视化学习率变化和训练效果

## 项目要求

- **难度**：简单
- **代码量**：< 200行
- **预计时间**：2-4小时
- **功能要求**：
  - 实现至少3种学习率调整策略
  - 在简单模型上测试
  - 绘制学习率曲线
  - 对比训练效果

## 实现步骤

1. **实现学习率调整策略**
   - 固定学习率
   - 指数衰减
   - 余弦退火
   - 阶梯衰减（可选）

2. **测试框架**
   - 使用相同的模型和数据集
   - 使用相同的优化器
   - 只改变学习率调整策略

3. **可视化**
   - 学习率变化曲线
   - 训练损失曲线对比
   - 最终性能对比
