# 简单项目1：优化算法对比工具

## 项目目标

创建一个工具，对比不同优化算法在相同问题上的表现。

## 项目概述

本项目将：
1. 实现多种优化算法（SGD、Momentum、Adam等）
2. 在相同问题上测试不同算法
3. 可视化对比结果

## 项目要求

- **难度**：简单
- **代码量**：< 200行
- **预计时间**：2-4小时
- **功能要求**：
  - 实现至少3种优化算法
  - 在简单优化问题上测试
  - 绘制损失曲线对比
  - 绘制优化路径对比

## 实现步骤

1. **实现优化算法**
   - SGD
   - Momentum
   - Adam

2. **测试问题**
   - 简单的二次函数优化
   - 或简单的神经网络训练

3. **可视化**
   - 损失曲线
   - 优化路径
   - 收敛速度对比

## 参考代码
