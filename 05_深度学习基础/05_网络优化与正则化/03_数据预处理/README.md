### 入门小测

**选择题**（每题2分，共10分）

1. 数据标准化的目的是？
   A. 增加数据量  B. 将数据缩放到相同尺度  C. 减少数据量  D. 改变数据分布
   **答案**：B
   **解释**：标准化将不同特征缩放到相同尺度，避免某些特征主导训练。

2. Z-score标准化的公式是？
   A. $x' = \frac{x - \min}{\max - \min}$  B. $x' = \frac{x - \mu}{\sigma}$  C. $x' = x \cdot \sigma$  D. $x' = x + \mu$
   **答案**：B
   **解释**：Z-score标准化使用均值和标准差：$x' = \frac{x - \mu}{\sigma}$。

3. Min-Max归一化的输出范围是？
   A. [0, 1]  B. [-1, 1]  C. (-∞, ∞)  D. [0, ∞)
   **答案**：A
   **解释**：Min-Max归一化将数据缩放到[0, 1]区间。

## 3. 核心知识点详解

### 3.1 数据预处理的重要性

#### 3.1.1 为什么需要预处理？

**问题**：
- **特征尺度不同**：不同特征的数值范围差异大（如年龄0-100，收入0-1000000）
- **训练不稳定**：尺度差异导致梯度更新不平衡
- **收敛慢**：需要更多迭代才能收敛

**解决方案**：数据预处理，将特征缩放到相同尺度。

### 3.2 标准化（Standardization）

#### 3.2.1 Z-score标准化

**公式**：
$$x' = \frac{x - \mu}{\sigma}$$

其中：
- $\mu$是均值
- $\sigma$是标准差

**特点**：
- 输出均值为0，标准差为1
- 保持原始分布形状
- 适合大多数情况

### 3.3 归一化（Normalization）

#### 3.3.1 Min-Max归一化

**公式**：
$$x' = \frac{x - x_{\min}}{x_{\max} - x_{\min}}$$

**特点**：
- 输出范围：[0, 1]
- 对异常值敏感
- 适合数据分布已知的情况

### 3.4 其他预处理方法

- **Robust标准化**：使用中位数和四分位距，对异常值鲁棒
- **L2归一化**：将向量归一化为单位向量
- **PCA降维**：减少特征维度
