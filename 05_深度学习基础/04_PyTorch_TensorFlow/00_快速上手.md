### 步骤2：运行第一个神经网络（PyTorch，10分钟）

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader, TensorDataset
import matplotlib.pyplot as plt

# 生成数据
X = torch.randn(100, 2)
y = (X[:, 0] + X[:, 1] > 0).float().unsqueeze(1)

# 定义模型（只需要几行代码！）
model = nn.Sequential(
    nn.Linear(2, 10),
    nn.ReLU(),
    nn.Linear(10, 1),
    nn.Sigmoid()
)

# 定义损失函数和优化器
criterion = nn.BCELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# 训练（只需要几行代码！）
for epoch in range(100):
    # 前向传播
    outputs = model(X)
    loss = criterion(outputs, y)
    
    # 反向传播
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    
    if epoch % 20 == 0:
        print(f'Epoch {epoch}, Loss: {loss.item():.4f}')

# 预测
with torch.no_grad():
    predictions = model(X)
    accuracy = ((predictions > 0.5) == y).float().mean()
    print(f'准确率: {accuracy.item():.2%}')

print("✅ 成功！你已经用PyTorch训练了一个神经网络！")
```

### 步骤3：运行第一个神经网络（TensorFlow，10分钟）

```python
import tensorflow as tf
from tensorflow import keras
import numpy as np

# 生成数据
X = np.random.randn(100, 2).astype(np.float32)
y = ((X[:, 0] + X[:, 1]) > 0).astype(np.float32)

# 定义模型（只需要几行代码！）
model = keras.Sequential([
    keras.layers.Dense(10, activation='relu', input_shape=(2,)),
    keras.layers.Dense(1, activation='sigmoid')
])

# 编译模型
model.compile(optimizer='sgd', loss='binary_crossentropy', metrics=['accuracy'])

# 训练（只需要一行代码！）
history = model.fit(X, y, epochs=100, verbose=0)

# 评估
loss, accuracy = model.evaluate(X, y, verbose=0)
print(f'准确率: {accuracy:.2%}')

print("✅ 成功！你已经用TensorFlow训练了一个神经网络！")
```

### 步骤4：可视化训练过程（5分钟）

```python
# PyTorch训练过程可视化
train_losses = []

for epoch in range(100):
    outputs = model(X)
    loss = criterion(outputs, y)
    
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    
    train_losses.append(loss.item())

# 可视化
plt.figure(figsize=(10, 6))
plt.plot(train_losses)
plt.xlabel('Epoch')
plt.ylabel('Loss')
plt.title('训练损失曲线')
plt.grid(True, alpha=0.3)
plt.show()
```

## 🎯 理解框架在做什么

### 直观理解

**深度学习框架就像"自动计算器"**：
1. **定义模型**：用几行代码定义网络结构
2. **自动梯度**：框架自动计算梯度（不需要手动实现反向传播）
3. **优化器**：自动更新参数
4. **训练循环**：只需要几行代码就能训练

**对比**：
- **从零实现**：需要几百行代码，手动实现所有细节
- **使用框架**：只需要几十行代码，框架处理所有细节

## ⚠️ 常见问题

**Q: 选择PyTorch还是TensorFlow？**
A: PyTorch更灵活，适合研究和实验；TensorFlow更适合生产环境。建议先学PyTorch。

**Q: 需要从零实现吗？**
A: 建议先理解原理（从零实现），再使用框架。这样能更好地理解框架在做什么。

**Q: 框架会自动处理所有事情吗？**
A: 框架处理梯度计算和参数更新，但你需要理解模型设计、损失函数选择等。
