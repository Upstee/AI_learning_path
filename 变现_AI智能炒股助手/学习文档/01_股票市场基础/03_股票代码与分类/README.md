# 03_股票代码与分类

## 📋 章节概述

本章节将介绍股票代码体系和股票分类方法，帮助你理解不同市场的股票代码规则，以及如何对股票进行分类，为后续的选股和分析打下基础。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解不同市场的股票代码体系
- 掌握股票的各种分类方法
- 了解全球主要股票市场的特点
- 能够识别和分类不同类型的股票

   - A股代码体系

   - 中国A股市场

   - 美国股市
   - 其他主要市场

## 📖 学习指南

### 学习顺序
1. 先学习"股票代码体系"，掌握代码规则
2. 再学习"股票分类方法"，理解分类逻辑
3. 最后学习"全球主要股票市场"，了解市场特点

### 学习方法
- **记忆理解**：股票代码需要记忆，但更重要的是理解规则
- **实际应用**：通过查看实际股票代码加深理解
- **对比学习**：对比不同市场的代码体系和分类方法

### 学习时间建议
- 总学习时间：2-3小时
- 股票代码体系：1小时
- 股票分类方法：1小时
- 全球主要股票市场：1小时
