# 01_股票市场基础

## 📋 模块概述

本模块是股票投资的基础入门模块，包含股票市场的基本概念、交易规则和股票分类等纯理论知识。这些知识是后续学习技术分析、基本面分析和量化交易的基础。

## 📚 章节列表

1. **[01_股票市场概述](./01_股票市场概述/README.md)** - 了解股票市场的基本概念和功能
2. **[02_股票交易基础](./02_股票交易基础/README.md)** - 学习股票交易的基本流程和规则
3. **[03_股票代码与分类](./03_股票代码与分类/README.md)** - 掌握股票代码体系和分类方法

## 📖 学习说明

本模块为**纯理论模块**，所有内容以Markdown文档（.md文件）形式呈现，不包含代码示例。每个章节包含：
- **README.md** - 章节学习指南和导论
- **知识点文件.md** - 详细的理论知识文档

如需图表辅助理解，可以使用Jupyter Notebook（.ipynb）文件，包含Markdown说明和Python绘图代码，直接放在相应的知识点文件夹下。
