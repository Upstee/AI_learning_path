# 02_股票交易基础

## 📋 章节概述

本章节将详细介绍股票交易的基本流程、规则制度和交易成本，帮助你理解如何实际进行股票交易，为后续的实战操作打下基础。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 掌握股票交易的完整流程
- 理解股票交易的基本规则和制度
- 了解交易费用和成本构成
- 具备进行股票交易的基础知识

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[股票交易流程](./股票交易流程.md)**
   - 开户流程
   - 下单方式
   - 交易执行流程
   - 结算与交割

2. **[交易规则与制度](./交易规则与制度.md)**
   - 交易时间
   - 涨跌停板制度
   - T+1交易制度
   - 交易单位与最小变动单位

3. **[交易费用与成本](./交易费用与成本.md)**
   - 佣金费用
   - 印花税
   - 过户费
   - 其他费用

## 📖 学习指南

### 学习顺序
1. 先学习"股票交易流程"，了解交易全流程
2. 再学习"交易规则与制度"，掌握交易规则
3. 最后学习"交易费用与成本"，了解交易成本

### 学习方法
- **理论理解**：理解每个环节的作用和意义
- **实际操作**：可以尝试模拟交易，加深理解
- **对比学习**：对比A股、港股、美股的交易规则差异

### 学习时间建议
- 总学习时间：3-4小时
- 股票交易流程：1-1.5小时
- 交易规则与制度：1-1.5小时
- 交易费用与成本：1小时
