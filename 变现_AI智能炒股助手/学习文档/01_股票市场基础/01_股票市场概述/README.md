# 01_股票市场概述

## 📋 章节概述

本章节将帮助你建立对股票市场的基础认知，理解股票市场的本质、功能和作用。这是学习股票投资的第一步，为后续的技术分析、基本面分析等深入学习打下坚实基础。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解股票市场的基本概念和本质
- 了解股票市场在经济发展中的功能和作用
- 建立对股票市场的整体认知框架

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[股票市场基本概念](./股票市场基本概念.md)**
   - 股票的定义和特征
   - 股票市场的基本概念
   - 股票市场的参与者

2. **[股票市场功能与作用](./股票市场功能与作用.md)**
   - 资源配置功能
   - 价格发现功能
   - 风险管理功能
   - 对经济发展的作用

## 📖 学习指南

### 学习顺序
1. 先学习"股票市场基本概念"，建立基础认知
2. 再学习"股票市场功能与作用"，理解市场价值

### 学习方法
- **理论理解**：仔细阅读每个知识点文档，理解核心概念
- **结合实际**：结合中国A股市场、美股市场等实际案例理解
- **思考总结**：每学完一个知识点，思考其在现实中的应用

### 学习时间建议
- 总学习时间：2-3小时
- 股票市场基本概念：1-1.5小时
- 股票市场功能与作用：1-1.5小时
