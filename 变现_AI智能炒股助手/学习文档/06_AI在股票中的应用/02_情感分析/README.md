# 02_情感分析

## 📋 章节概述

情感分析通过分析文本中的情感倾向来预测市场情绪，本章节将介绍文本情感分析、新闻情感分析和社交媒体情感分析在股票市场中的应用。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解情感分析在股票市场中的价值
- 掌握文本情感分析的基本方法
- 掌握新闻情感分析的应用
- 掌握社交媒体情感分析的应用
- 能够构建情感分析系统

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[文本情感分析](./文本情感分析.md)**
   - 情感分析基础
   - 词典方法
   - 机器学习方法
   - 深度学习方法

2. **[新闻情感分析](./新闻情感分析.md)**
   - 新闻数据获取
   - 新闻情感提取
   - 情感与股价关系
   - 新闻情感指标构建

3. **[社交媒体情感分析](./社交媒体情感分析.md)**
   - 社交媒体数据获取
   - 用户情感分析
   - 情感传播分析
   - 社交媒体情绪指标

## 📖 学习指南

### 学习顺序
1. 先学习"文本情感分析"，掌握基础方法
2. 再学习"新闻情感分析"，了解新闻应用
3. 最后学习"社交媒体情感分析"，了解社交媒体应用

### 学习方法
- **理论理解**：理解情感分析的NLP技术
- **编程实践**：通过代码实现情感分析模型
- **数据获取**：掌握文本数据的获取和清洗方法
- **应用验证**：验证情感分析对股价的预测能力

### 学习时间建议
- 总学习时间：6-8小时
- 文本情感分析：2-2.5小时
- 新闻情感分析：2-2.5小时
- 社交媒体情感分析：2-3小时
