# 01_预测模型

## 📋 章节概述

预测模型是AI在股票分析中的核心应用，本章节将介绍时间序列预测、机器学习和深度学习在股价预测中的应用，帮助你掌握构建股票预测模型的方法。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解股票价格预测的挑战和特点
- 掌握时间序列预测模型的应用
- 掌握机器学习预测模型的应用
- 掌握深度学习预测模型的应用
- 能够构建完整的股票预测系统

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[时间序列预测模型](./时间序列预测模型.md)**
   - ARIMA模型
   - GARCH模型
   - 状态空间模型
   - 时间序列特征工程

2. **[机器学习预测模型](./机器学习预测模型.md)**
   - 回归模型应用
   - 集成学习方法
   - 特征选择与工程
   - 模型评估方法

3. **[深度学习预测模型](./深度学习预测模型.md)**
   - LSTM模型
   - GRU模型
   - Transformer模型
   - 注意力机制应用

## 📖 学习指南

### 学习顺序
1. 先学习"时间序列预测模型"，掌握传统方法
2. 再学习"机器学习预测模型"，掌握现代方法
3. 最后学习"深度学习预测模型"，掌握前沿方法

### 学习方法
- **理论理解**：理解各种预测模型的原理
- **编程实践**：通过代码实现预测模型
- **数据准备**：掌握股票数据的预处理方法
- **模型对比**：对比不同模型的预测效果

### 学习时间建议
- 总学习时间：8-10小时
- 时间序列预测模型：2.5-3小时
- 机器学习预测模型：2.5-3小时
- 深度学习预测模型：3-4小时
