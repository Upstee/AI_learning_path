# 04_强化学习

## 📋 章节概述

强化学习可以通过与环境交互来学习最优交易策略，本章节将介绍强化学习在股票交易中的应用，帮助你掌握如何构建基于强化学习的交易系统。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解强化学习的基本原理
- 掌握交易环境的建模方法
- 掌握强化学习策略的训练方法
- 能够构建基于强化学习的交易系统

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[强化学习基础](./强化学习基础.md)**
   - 强化学习基本概念
   - 马尔可夫决策过程（MDP）
   - 价值函数与策略
   - 主要算法（Q-learning、DQN、PPO等）

2. **[交易策略强化学习](./交易策略强化学习.md)**
   - 交易环境建模
   - 状态空间设计
   - 动作空间设计
   - 奖励函数设计
   - 策略训练与优化

## 📖 学习指南

### 学习顺序
1. 先学习"强化学习基础"，掌握理论基础
2. 再学习"交易策略强化学习"，掌握实际应用

### 学习方法
- **理论理解**：深入理解强化学习的数学原理
- **环境建模**：学会将交易问题转化为强化学习问题
- **编程实践**：通过代码实现强化学习交易系统
- **参数调优**：掌握强化学习模型的调参技巧

### 学习时间建议
- 总学习时间：6-8小时
- 强化学习基础：3-4小时
- 交易策略强化学习：3-4小时
