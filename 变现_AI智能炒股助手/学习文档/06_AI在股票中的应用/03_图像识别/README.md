# 03_图像识别

## 📋 章节概述

图像识别技术可以用于识别K线图形态和图表模式，本章节将介绍如何利用计算机视觉技术自动识别技术分析中的图表模式，提高技术分析的效率。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解图像识别在股票分析中的应用价值
- 掌握图表模式识别的方法
- 掌握K线形态自动识别技术
- 能够构建图表识别系统

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[图表模式识别](./图表模式识别.md)**
   - 图表图像预处理
   - 特征提取方法
   - 模式识别算法
   - CNN在图表识别中的应用

2. **[K线形态识别](./K线形态识别.md)**
   - K线图像数据准备
   - K线形态特征提取
   - 形态分类模型
   - 实时形态识别系统

## 📖 学习指南

### 学习顺序
1. 先学习"图表模式识别"，掌握基础方法
2. 再学习"K线形态识别"，掌握具体应用

### 学习方法
- **理论理解**：理解计算机视觉的基本原理
- **编程实践**：通过代码实现图像识别模型
- **数据准备**：掌握图表图像的预处理方法
- **模型训练**：学会训练和优化识别模型

### 学习时间建议
- 总学习时间：5-6小时
- 图表模式识别：2.5-3小时
- K线形态识别：2.5-3小时
