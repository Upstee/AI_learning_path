## 1. 需求理论

### 1.1 需求的定义

**需求（Demand）**是指消费者在某一时期内，在各种可能的价格水平下，愿意并且能够购买的商品或服务的数量。

需求包含两个要素：
- **购买意愿**：消费者想要购买
- **购买能力**：消费者有支付能力

### 1.2 需求定律

**需求定律（Law of Demand）**：在其他条件不变的情况下，商品的价格与需求量之间呈反方向变动关系。

- 价格上升 → 需求量减少
- 价格下降 → 需求量增加

### 1.3 影响需求的因素

1. **商品自身价格**（沿需求曲线移动）
2. **消费者收入**
   - 正常商品：收入增加，需求增加
   - 劣等商品：收入增加，需求减少
3. **相关商品价格**
   - 替代品：价格上升，需求增加
   - 互补品：价格上升，需求减少
4. **消费者偏好**
5. **预期价格**
6. **消费者数量**

### 1.4 需求函数

需求函数表示需求量与影响需求的各种因素之间的关系：

```
Qd = f(P, I, Pr, T, Pe, N, ...)
```

其中：
- Qd：需求量
- P：商品价格
- I：消费者收入
- Pr：相关商品价格
- T：消费者偏好
- Pe：预期价格
- N：消费者数量

## 2. 供给理论

### 2.1 供给的定义

**供给（Supply）**是指生产者在某一时期内，在各种可能的价格水平下，愿意并且能够提供的商品或服务的数量。

供给包含两个要素：
- **提供意愿**：生产者愿意提供
- **提供能力**：生产者有生产能力

### 2.2 供给定律

**供给定律（Law of Supply）**：在其他条件不变的情况下，商品的价格与供给量之间呈正方向变动关系。

- 价格上升 → 供给量增加
- 价格下降 → 供给量减少

### 2.3 影响供给的因素

1. **商品自身价格**（沿供给曲线移动）
2. **生产成本**
   - 原材料价格
   - 工资水平
   - 技术水平
3. **相关商品价格**
   - 替代品：价格上升，供给减少
   - 互补品：价格上升，供给增加
4. **生产技术**
5. **生产者预期**
6. **生产者数量**
7. **政府政策**（税收、补贴等）

### 2.4 供给函数

供给函数表示供给量与影响供给的各种因素之间的关系：

```
Qs = f(P, C, Pr, Tech, Pe, N, G, ...)
```

其中：
- Qs：供给量
- P：商品价格
- C：生产成本
- Pr：相关商品价格
- Tech：技术水平
- Pe：生产者预期
- N：生产者数量
- G：政府政策

## 3. 市场均衡

### 3.1 均衡价格与均衡数量

**市场均衡（Market Equilibrium）**是指需求量和供给量相等时的状态。

- **均衡价格（Equilibrium Price）**：使需求量等于供给量的价格
- **均衡数量（Equilibrium Quantity）**：在均衡价格下的交易量

### 3.2 均衡的形成

当市场价格偏离均衡价格时，市场力量会自动调整：

- **价格高于均衡价格**：
  - 供给量 > 需求量 → 供过于求 → 价格下降 → 趋向均衡
  
- **价格低于均衡价格**：
  - 需求量 > 供给量 → 供不应求 → 价格上升 → 趋向均衡

### 3.3 均衡的变动

**需求变动**（需求曲线移动）：
- 需求增加 → 均衡价格上升，均衡数量增加
- 需求减少 → 均衡价格下降，均衡数量减少

**供给变动**（供给曲线移动）：
- 供给增加 → 均衡价格下降，均衡数量增加
- 供给减少 → 均衡价格上升，均衡数量减少

## 4. 供需理论在股票市场中的应用

### 4.1 股票市场的供需关系

在股票市场中：
- **需求方**：投资者（买入股票）
- **供给方**：股东（卖出股票）
- **价格**：股票价格
- **数量**：交易量

### 4.2 影响股票需求的因素

1. **股票价格**（需求定律）
2. **投资者收入**
3. **预期收益**
4. **风险偏好**
5. **替代投资品**（债券、房地产等）
6. **市场情绪**

### 4.3 影响股票供给的因素

1. **股票价格**（供给定律）
2. **公司基本面**
3. **市场流动性**
4. **股东减持意愿**
5. **新股发行**
6. **政策环境**

### 4.4 股票市场均衡

- **均衡价格**：买卖双方达成一致的价格
- **均衡数量**：在该价格下的成交量
- **价格发现**：市场通过交易不断寻找均衡价格

---

## 5. 关键概念总结

| 概念 | 定义 | 股票市场应用 |
|------|------|-------------|
| 需求 | 消费者愿意且能够购买的数量 | 投资者愿意买入的股票数量 |
| 供给 | 生产者愿意且能够提供的数量 | 股东愿意卖出的股票数量 |
| 需求定律 | 价格与需求量反方向变动 | 股价越高，买入意愿越低 |
| 供给定律 | 价格与供给量正方向变动 | 股价越高，卖出意愿越高 |
| 市场均衡 | 需求量等于供给量 | 买卖双方达成一致的价格和数量 |
