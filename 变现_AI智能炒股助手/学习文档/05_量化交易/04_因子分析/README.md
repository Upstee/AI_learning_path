# 04_因子分析

## 📋 章节概述

因子分析是量化选股和策略构建的基础，本章节将帮助你掌握如何挖掘有效因子，并验证因子的有效性，为构建多因子模型打下基础。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解因子的概念和分类
- 掌握因子挖掘的方法
- 掌握因子有效性检验方法
- 能够构建多因子模型

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[因子挖掘方法](./因子挖掘方法.md)**
   - 因子来源
   - 基本面因子
   - 技术面因子
   - 另类因子
   - 因子构建方法

2. **[因子有效性检验](./因子有效性检验.md)**
   - IC分析
   - 分层回测
   - 因子收益率分析
   - 因子稳定性检验
   - 因子相关性分析

## 📖 学习指南

### 学习顺序
1. 先学习"因子挖掘方法"，掌握如何发现因子
2. 再学习"因子有效性检验"，验证因子有效性

### 学习方法
- **理论理解**：理解因子的经济学和金融学含义
- **数据挖掘**：通过数据分析挖掘新因子
- **实证验证**：通过历史数据验证因子有效性
- **组合构建**：学会将多个因子组合使用

### 学习时间建议
- 总学习时间：5-6小时
- 因子挖掘方法：2.5-3小时
- 因子有效性检验：2.5-3小时
