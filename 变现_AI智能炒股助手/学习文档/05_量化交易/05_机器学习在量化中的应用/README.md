# 05_机器学习在量化中的应用

## 📋 章节概述

机器学习为量化交易提供了强大的工具，本章节将介绍监督学习、深度学习和强化学习在量化交易中的应用，帮助你掌握AI技术在量化交易中的使用方法。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解机器学习在量化交易中的应用场景
- 掌握监督学习在量化中的应用方法
- 掌握深度学习在量化中的应用方法
- 了解强化学习在量化交易中的应用

1. 先学习"监督学习在量化中的应用"，掌握基础应用
