# 03_风险管理

## 📋 章节概述

风险管理是量化交易成功的关键，本章节将帮助你掌握风险度量和风险控制的方法，学会在交易中有效管理风险。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解风险的本质和类型
- 掌握各种风险度量方法
- 掌握风险控制策略
- 能够构建完整的风险管理体系

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[风险度量方法](./风险度量方法.md)**
   - 波动率（Volatility）
   - 最大回撤（Max Drawdown）
   - VaR（风险价值）
   - CVaR（条件风险价值）
   - 夏普比率
   - 索提诺比率

2. **[风险控制策略](./风险控制策略.md)**
   - 仓位管理
   - 止损策略
   - 风险预算
   - 组合风险管理
   - 实时风险监控

## 📖 学习指南

### 学习顺序
1. 先学习"风险度量方法"，理解如何量化风险
2. 再学习"风险控制策略"，掌握风险控制手段

### 学习方法
- **理论理解**：理解风险度量的数学原理
- **实际计算**：通过实际案例计算各种风险指标
- **策略设计**：设计风险控制策略
- **系统思维**：建立完整的风险管理体系

### 学习时间建议
- 总学习时间：4-5小时
- 风险度量方法：2-2.5小时
- 风险控制策略：2-2.5小时
