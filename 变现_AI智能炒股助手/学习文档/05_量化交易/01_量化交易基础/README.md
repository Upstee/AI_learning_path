# 01_量化交易基础

## 📋 章节概述

量化交易是利用数学模型和计算机技术进行交易决策的方法，本章节将帮助你建立对量化交易的整体认知，理解量化交易系统的基本架构。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解量化交易的基本概念和特点
- 了解量化交易的发展历程
- 掌握量化交易系统的基本架构
- 理解量化交易与传统交易的区别

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[量化交易概述](./量化交易概述.md)**
   - 量化交易的定义
   - 量化交易的特点
   - 量化交易的优势与局限
   - 量化交易的发展历程

2. **[量化交易系统架构](./量化交易系统架构.md)**
   - 数据获取模块
   - 策略研究模块
   - 回测模块
   - 实盘交易模块
   - 风险控制模块

## 📖 学习指南

### 学习顺序
1. 先学习"量化交易概述"，建立基础认知
2. 再学习"量化交易系统架构"，理解系统组成

### 学习方法
- **理论理解**：理解量化交易的核心思想
- **系统认知**：理解量化交易系统的完整流程
- **工具了解**：了解常用的量化交易平台和工具

### 学习时间建议
- 总学习时间：3-4小时
- 量化交易概述：1.5-2小时
- 量化交易系统架构：1.5-2小时
