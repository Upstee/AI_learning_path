# 03_图表形态

## 📋 章节概述

图表形态是技术分析中识别趋势转折和延续的重要方法，本章节将帮助你掌握各种反转形态和持续形态的识别技巧，提高对市场走势的判断能力。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 识别常见的反转形态
- 识别常见的持续形态
- 理解形态形成的市场心理
- 掌握形态的目标位测算方法
- 能够通过形态判断买卖时机

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[反转形态](./反转形态.md)**
   - 头肩顶/底
   - 双顶/双底
   - 三重顶/底
   - V型反转
   - 圆弧顶/底

2. **[持续形态](./持续形态.md)**
   - 三角形形态
   - 旗形形态
   - 楔形形态
   - 矩形形态
   - 菱形形态

## 📖 学习指南

### 学习顺序
1. 先学习"反转形态"，掌握趋势转折的判断
2. 再学习"持续形态"，掌握趋势延续的判断

### 学习方法
- **图形记忆**：通过大量图形记忆各种形态特征
- **实践识别**：在真实图表中寻找和验证形态
- **量价配合**：结合成交量和价格走势确认形态

### 学习时间建议
- 总学习时间：4-5小时
- 反转形态：2-2.5小时
- 持续形态：2-2.5小时
