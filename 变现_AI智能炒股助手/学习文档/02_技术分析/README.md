# 02_技术分析

## 📋 模块概述

本模块介绍技术分析的理论和方法，包括K线图、技术指标、图表形态和交易策略。由于技术分析需要大量图表辅助理解，本模块会使用Jupyter Notebook（.ipynb）文件，结合Markdown说明和Python绘图代码来呈现内容。

## 📚 章节列表

1. **[01_K线图基础](./01_K线图基础/README.md)** - 学习K线图的基本构成和形态识别
2. **[02_技术指标](./02_技术指标/README.md)** - 掌握各类技术指标的原理和应用
3. **[03_图表形态](./03_图表形态/README.md)** - 识别反转形态和持续形态
4. **[04_技术分析策略](./04_技术分析策略/README.md)** - 构建基于技术分析的交易策略
   - **重要**：包含市场心理与多层次博弈思维，这是技术分析的高级应用

## 📖 学习说明

本模块内容以**理论+图表**的形式呈现：
- **README.md** - 章节学习指南和导论
- **知识点文件.md** - 理论知识文档（纯文字说明）
- **知识点文件.ipynb** - 图表辅助学习（Markdown说明+Python绘图代码）

**注意**：对于需要图表辅助的知识点，会使用.ipynb文件替代或补充.md文件，直接在知识点文件夹下，不单独设置"代码示例"文件夹。
