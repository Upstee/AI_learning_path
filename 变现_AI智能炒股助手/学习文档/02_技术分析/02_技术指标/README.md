# 02_技术指标

## 📋 章节概述

技术指标是技术分析的核心工具，本章节将介绍各类技术指标的原理和应用方法，帮助你掌握趋势分析、震荡分析和成交量分析的技术手段。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解各类技术指标的计算原理
- 掌握趋势指标的使用方法
- 掌握震荡指标的使用方法
- 理解成交量指标的市场含义
- 能够综合运用多种指标进行分析

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[趋势指标](./趋势指标.md)**
   - 移动平均线（MA）
   - 指数移动平均线（EMA）
   - MACD指标
   - 布林带（BOLL）

2. **[震荡指标](./震荡指标.md)**
   - 相对强弱指标（RSI）
   - 随机指标（KDJ）
   - 威廉指标（WR）
   - 商品通道指数（CCI）

3. **[成交量指标](./成交量指标.md)**
   - 成交量（VOL）
   - 量价关系
   - OBV指标
   - 成交量移动平均

## 📖 学习指南

### 学习顺序
1. 先学习"趋势指标"，掌握趋势分析方法
2. 再学习"震荡指标"，掌握震荡市场分析方法
3. 最后学习"成交量指标"，理解量价关系

### 学习方法
- **理论理解**：理解每个指标的计算公式和市场含义
- **参数设置**：了解不同参数设置对指标的影响
- **综合应用**：学会将多个指标结合使用，避免单一指标的局限性

### 学习时间建议
- 总学习时间：5-6小时
- 趋势指标：2小时
- 震荡指标：2小时
- 成交量指标：1-2小时
