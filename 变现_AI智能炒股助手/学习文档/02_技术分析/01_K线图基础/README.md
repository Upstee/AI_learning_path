# 01_K线图基础

## 📋 章节概述

K线图是技术分析的基础工具，本章节将帮助你掌握K线图的基本构成和常见K线形态的识别方法，这是进行技术分析的第一步。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解K线图的基本构成要素
- 识别常见的K线形态
- 理解不同K线形态的市场含义
- 能够通过K线图初步判断市场趋势

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[K线图基本构成](./K线图基本构成.ipynb)** ⭐ 使用Jupyter Notebook，可以边学边运行
   - K线的四个价格要素
   - K线的绘制方法
   - 不同周期的K线图
   - K线图的阅读方法
   - **学习方式**：打开.ipynb文件，运行代码单元格，观察图表结果

2. **[K线形态识别](./K线形态识别.md)**
   - 单根K线形态
   - 双根K线组合
   - 多根K线组合
   - 特殊K线形态

## 📖 学习指南

### 学习顺序
1. 先学习"K线图基本构成"，理解K线的基础知识
2. 再学习"K线形态识别"，掌握形态分析方法

### 学习方法
- **理论理解**：理解K线的构成和市场含义
- **图形记忆**：通过大量图形记忆不同K线形态
- **实践应用**：在真实K线图上识别和验证形态

### 学习时间建议
- 总学习时间：3-4小时
- K线图基本构成：1.5-2小时
- K线形态识别：1.5-2小时
