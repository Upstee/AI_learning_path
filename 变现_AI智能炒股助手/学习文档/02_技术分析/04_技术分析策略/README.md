# 04_技术分析策略

## 📋 章节概述

本章节将介绍基于技术分析的各种交易策略，帮助你将前面学到的K线、指标、形态等知识综合运用，形成完整的交易策略体系。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解不同市场环境下的策略选择
- 掌握趋势跟踪策略的构建方法
- 掌握均值回归策略的构建方法
- 掌握突破策略的构建方法
- 能够根据市场情况选择合适的策略

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[市场心理与博弈](./市场心理与博弈.ipynb)** ⭐ 使用Jupyter Notebook，理解市场心理和多层次博弈思维
   - 市场心理的重要性
   - 多层次博弈思维（一层、二层、三层）
   - 市场情绪指标
   - 博弈思维的实际应用
   - 综合博弈策略

2. **[趋势跟踪策略](./趋势跟踪策略.md)**
   - 策略原理
   - 入场和出场信号
   - 止损和止盈设置
   - 策略优化方法

3. **[均值回归策略](./均值回归策略.md)**
   - 策略原理
   - 超买超卖判断
   - 入场和出场时机
   - 风险控制方法

4. **[突破策略](./突破策略.md)**
   - 突破识别方法
   - 真假突破判断
   - 突破策略执行
   - 失败突破处理

## 📖 学习指南

### 学习顺序
1. **首先学习"市场心理与博弈"**，这是技术分析的高级思维，理解市场参与者的心理和多层次博弈关系
2. 再学习"趋势跟踪策略"，适合趋势明显的市场
3. 然后学习"均值回归策略"，适合震荡市场
4. 最后学习"突破策略"，适合关键位置突破

### 学习方法
- **策略理解**：理解每种策略的适用场景和原理
- **回测验证**：通过历史数据回测验证策略有效性
- **风险控制**：重点学习止损和风险控制方法
- **综合运用**：学会根据市场情况灵活切换策略

### 学习时间建议
- 总学习时间：7-8小时
- 市场心理与博弈：2-3小时（重要，需要深入理解）
- 趋势跟踪策略：2小时
- 均值回归策略：2小时
- 突破策略：1-2小时
