# 01_财务报表分析

## 📋 章节概述

财务报表是了解公司经营状况的核心资料，本章节将帮助你掌握三大财务报表的分析方法，理解财务比率的含义，为基本面分析和公司估值打下基础。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解三大财务报表的结构和内容
- 掌握资产负债表分析方法
- 掌握利润表分析方法
- 掌握现金流量表分析方法
- 理解主要财务比率的含义和应用

本章节包含以下知识点，建议按顺序学习：

1. **[资产负债表分析](./资产负债表分析.md)**
   - 资产负债表结构
   - 资产质量分析
   - 负债结构分析
   - 所有者权益分析

2. **[利润表分析](./利润表分析.md)**
   - 利润表结构
   - 收入质量分析
   - 成本费用分析
   - 盈利能力分析

3. **[现金流量表分析](./现金流量表分析.md)**
   - 现金流量表结构
   - 经营活动现金流分析
   - 投资活动现金流分析
   - 筹资活动现金流分析

4. **[财务比率分析](./财务比率分析.md)**
   - 盈利能力比率
   - 偿债能力比率

## 📖 学习指南

### 学习顺序
1. 先学习三大报表分析（资产负债表、利润表、现金流量表）
2. 再学习财务比率分析，综合运用三大报表数据

### 学习方法
- **理论理解**：理解每个报表项目的含义
- **实际案例**：通过真实公司财报进行练习
- **对比分析**：对比同行业不同公司的财务数据
- **趋势分析**：分析公司多年财务数据的变化趋势

### 学习时间建议
- 总学习时间：6-8小时
- 资产负债表分析：1.5-2小时
- 利润表分析：1.5-2小时
- 现金流量表分析：1.5-2小时
- 财务比率分析：1.5-2小时
