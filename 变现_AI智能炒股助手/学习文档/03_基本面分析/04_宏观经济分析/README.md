# 04_宏观经济分析

## 📋 章节概述

宏观经济环境对股票市场有重要影响，本章节将帮助你理解货币政策、财政政策和经济周期对股市的影响，掌握宏观经济分析的方法。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解货币政策对股市的影响机制
- 理解财政政策对股市的影响机制
- 理解经济周期与股市的关系
- 能够通过宏观经济分析判断市场趋势

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[货币政策分析](./货币政策分析.md)**
   - 货币政策工具
   - 利率对股市的影响
   - 货币供应量对股市的影响
   - 央行政策解读

2. **[财政政策分析](./财政政策分析.md)**
   - 财政政策工具
   - 财政支出对股市的影响
   - 税收政策对股市的影响
   - 财政政策效果分析

3. **[经济周期分析](./经济周期分析.md)**
   - 经济周期阶段
   - 经济指标解读
   - 经济周期与股市周期
   - 周期投资策略

## 📖 学习指南

### 学习顺序
1. 先学习"货币政策分析"，理解货币环境的影响
2. 再学习"财政政策分析"，理解财政政策的影响
3. 最后学习"经济周期分析"，综合理解宏观经济

### 学习方法
- **理论理解**：理解宏观经济政策的作用机制
- **数据跟踪**：关注重要的宏观经济数据发布
- **政策解读**：学会解读央行和政府的政策信号
- **历史回顾**：回顾历史政策对股市的影响

### 学习时间建议
- 总学习时间：5-6小时
- 货币政策分析：2小时
- 财政政策分析：1.5-2小时
- 经济周期分析：1.5-2小时
