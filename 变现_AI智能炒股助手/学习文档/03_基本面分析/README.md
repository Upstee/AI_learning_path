# 03_基本面分析

## 📋 模块概述

本模块介绍基本面分析的理论和方法，包括财务报表分析、公司估值、行业分析和宏观经济分析。部分内容（如财务图表、估值模型）会使用Jupyter Notebook文件来辅助理解。

## 📚 章节列表

1. **[01_财务报表分析](./01_财务报表分析/README.md)** - 掌握三大财务报表的分析方法
2. **[02_公司估值](./02_公司估值/README.md)** - 学习绝对估值法和相对估值法
3. **[03_行业分析](./03_行业分析/README.md)** - 理解行业生命周期和竞争分析
4. **[04_宏观经济分析](./04_宏观经济分析/README.md)** - 分析宏观经济政策对股市的影响

## 📖 学习说明

本模块内容以**理论为主，图表辅助**的形式呈现：
- **README.md** - 章节学习指南和导论
- **知识点文件.md** - 理论知识文档（纯文字说明）
- **知识点文件.ipynb** - 图表辅助学习（如财务图表、估值模型可视化等，可选）

**注意**：对于需要图表辅助的知识点，会使用.ipynb文件，直接在知识点文件夹下，不单独设置"代码示例"文件夹。
