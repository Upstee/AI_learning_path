# 02_公司估值

## 📋 章节概述

公司估值是基本面分析的核心，本章节将介绍绝对估值法和相对估值法，帮助你掌握如何判断公司的内在价值，为投资决策提供依据。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解公司估值的核心原理
- 掌握绝对估值法的应用
- 掌握相对估值法的应用
- 理解不同估值方法的适用场景
- 能够对公司进行合理估值

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[绝对估值法](./绝对估值法.md)**
   - 现金流折现模型（DCF）
   - 股利折现模型（DDM）
   - 自由现金流模型
   - 估值参数确定

2. **[相对估值法](./相对估值法.md)**
   - 市盈率（PE）
   - 市净率（PB）
   - 市销率（PS）
   - PEG比率
   - EV/EBITDA

## 📖 学习指南

### 学习顺序
1. 先学习"绝对估值法"，理解估值的理论基础
2. 再学习"相对估值法"，掌握实用的估值方法

### 学习方法
- **理论理解**：理解估值模型的理论基础
- **实际计算**：通过实际案例进行估值计算
- **方法对比**：对比不同估值方法的结果
- **行业应用**：了解不同行业适用的估值方法

### 学习时间建议
- 总学习时间：5-6小时
- 绝对估值法：2.5-3小时
- 相对估值法：2.5-3小时
