# 03_行业分析

## 📋 章节概述

行业分析是基本面分析的重要组成部分，本章节将帮助你理解行业生命周期和竞争格局，掌握行业分析的方法，为选股和投资决策提供参考。

## 🎯 学习目标

通过本章节的学习，你将能够：
- 理解行业生命周期的各个阶段
- 掌握行业竞争分析方法
- 识别行业发展的关键驱动因素
- 能够判断行业的投资价值

## 📚 知识点列表

本章节包含以下知识点，建议按顺序学习：

1. **[行业生命周期](./行业生命周期.md)**
   - 初创期
   - 成长期
   - 成熟期
   - 衰退期
   - 各阶段特征和投资策略

2. **[行业竞争分析](./行业竞争分析.md)**
   - 波特五力模型
   - 行业集中度分析
   - 行业壁垒分析
   - 行业盈利能力分析

## 📖 学习指南

### 学习顺序
1. 先学习"行业生命周期"，理解行业发展规律
2. 再学习"行业竞争分析"，掌握竞争格局分析方法

### 学习方法
- **理论理解**：理解行业分析的理论框架
- **案例分析**：分析不同行业的实际案例
- **对比分析**：对比不同行业的发展阶段和竞争格局
- **跟踪研究**：持续跟踪关注行业的发展变化

### 学习时间建议
- 总学习时间：4-5小时
- 行业生命周期：2-2.5小时
- 行业竞争分析：2-2.5小时
