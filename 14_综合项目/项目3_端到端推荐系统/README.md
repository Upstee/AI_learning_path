## 📚 课程结构

本课程包含以下8个部分：

1. **快速上手**：30分钟快速体验
2. **理论笔记**：详细的理论讲解和数学推导
3. **代码示例**：交互式Jupyter Notebook示例
4. **练习题**：基础、进阶、挑战三个层次的练习
5. **实战案例**：8个实战项目（2简单+2中等+2进阶+2综合）
6. **学习检查点**：自我评估和知识检查清单
7. **常见问题FAQ**：常见问题解答
8. **自我评估**：概念题、编程题、项目评估
